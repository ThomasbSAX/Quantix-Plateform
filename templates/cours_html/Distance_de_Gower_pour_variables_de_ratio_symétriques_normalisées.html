La distance de Gower pour variables de ratio symétriques normalisées

Quantix

2026-01-11

Introduction et Motivations

L’analyse des données multidimensionnelles est un enjeu fondamental dans de nombreux domaines scientifiques. Parmi les outils essentiels pour cette analyse, les distances entre objets jouent un rôle central. La distance de Gower, introduite par Michael J. Gower en 1971, est une mesure de dissimilarité particulièrement adaptée aux données mixtes, c’est-à-dire celles qui combinent des variables de différents types.

Dans ce chapitre, nous nous concentrons sur une variante spécifique de la distance de Gower, adaptée aux variables de ratio symétriques normalisées. Ces variables sont celles pour lesquelles les valeurs nulles n’ont pas de sens et qui peuvent être comparées de manière proportionnelle. La normalisation symétrique permet de standardiser ces variables de manière à ce qu’elles soient comparables sur une échelle commune.

L’importance de cette distance réside dans sa capacité à traiter des données complexes tout en préservant les propriétés métriques essentielles. Elle est indispensable dans des domaines tels que la biologie, l’économie et les sciences sociales, où les données sont souvent de nature mixte.

Définitions

Pour définir la distance de Gower pour des variables de ratio symétriques normalisées, commençons par comprendre les concepts clés.

Variables de Ratio Symétriques

Une variable de ratio est une variable quantifiable pour laquelle les valeurs nulles n’ont pas de sens et qui peut être exprimée en termes de proportions. Par exemple, la taille ou le poids sont des variables de ratio.

Une variable de ratio symétrique est une variable de ratio pour laquelle les valeurs peuvent être comparées de manière proportionnelle. Par exemple, si une personne pèse deux fois plus qu’une autre, cette relation est significative.

Formellement, soit \(X\) une variable de ratio symétrique. On peut écrire :

\[X : \Omega \rightarrow \mathbb{R}^+\]

où \(\Omega\) est l’ensemble des objets et \(\mathbb{R}^+\) est l’ensemble des nombres réels positifs.

Normalisation Symétrique

La normalisation symétrique consiste à transformer une variable de ratio \(X\) en une variable \(X'\) telle que :

\[X' = \frac{X}{\max(X)}\]

où \(\max(X)\) est la valeur maximale de \(X\).

Cette normalisation permet de comparer les variables sur une échelle commune, allant de 0 à 1.

Distance de Gower

La distance de Gower pour des variables de ratio symétriques normalisées est une mesure de dissimilarité entre deux objets \(i\) et \(j\) basée sur les différences normalisées des variables.

Pour deux objets \(i\) et \(j\), la distance de Gower \(d_{ij}\) est définie comme :

\[d_{ij} = \frac{\sum_{k=1}^{p} s_{ijk} \delta_{ijk}}{\sum_{k=1}^{p} s_{ijk}}\]

où : - \(p\) est le nombre de variables, - \(s_{ijk}\) est un indicateur binaire qui vaut 1 si la variable \(k\) est définie pour les objets \(i\) et \(j\), et 0 sinon, - \(\delta_{ijk}\) est la dissimilarité entre les objets \(i\) et \(j\) pour la variable \(k\).

Pour des variables de ratio symétriques normalisées, la dissimilarité \(\delta_{ijk}\) est définie comme :

\[\delta_{ijk} = |x'_{ik} - x'_{jk}|\]

où \(x'_{ik}\) et \(x'_{jk}\) sont les valeurs normalisées des objets \(i\) et \(j\) pour la variable \(k\).

Théorèmes

Théorème de Gower

Le théorème de Gower stipule que la distance de Gower est une mesure de dissimilarité valide, c’est-à-dire qu’elle satisfait les propriétés suivantes :

Soit \(d_{ij}\) la distance de Gower entre deux objets \(i\) et \(j\). Alors :

  1. \(d_{ij} \geq 0\) (non-négativité),

  2. \(d_{ij} = 0\) si et seulement si \(i = j\) (identité),

  3. \(d_{ij} = d_{ji}\) (symétrie),

  4. \(d_{ik} \leq d_{ij} + d_{jk}\) (inégalité triangulaire).

Preuve du Théorème de Gower

Pour prouver le théorème de Gower, nous allons vérifier chacune des propriétés énoncées.

Proof.

  1. Non-négativité : Puisque \(\delta_{ijk} = |x'_{ik} - x'_{jk}| \geq 0\), il s’ensuit que \(d_{ij} \geq 0\).

  2. Identité : Si \(i = j\), alors \(x'_{ik} = x'_{jk}\) pour tout \(k\), donc \(\delta_{ijk} = 0\). Par conséquent, \(d_{ij} = 0\).

  3. Symétrie : Puisque \(\delta_{ijk} = |x'_{ik} - x'_{jk}| = |x'_{jk} - x'_{ik}| = \delta_{jik}\), il s’ensuit que \(d_{ij} = d_{ji}\).

  4. Inégalité triangulaire : Pour tout \(k\), on a \(\delta_{ik} \leq \delta_{ij} + \delta_{jk}\) en vertu de l’inégalité triangulaire pour la distance euclidienne. En sommant sur \(k\) et en divisant par le nombre de variables définies, on obtient \(d_{ik} \leq d_{ij} + d_{jk}\).

Propriétés et Corollaires

Propriété 1 : Invariance par Normalisation

La distance de Gower est invariante par normalisation symétrique. Cela signifie que si toutes les variables sont normalisées de la même manière, la distance reste inchangée.

Soit \(X\) une variable de ratio symétrique et \(X'\) sa version normalisée. Alors, pour deux objets \(i\) et \(j\), on a :

\[d_{ij}(X) = d_{ij}(X')\]

où \(d_{ij}(X)\) et \(d_{ij}(X')\) sont les distances de Gower calculées respectivement sur \(X\) et \(X'\).

Preuve de l’Invariance par Normalisation

Proof. La normalisation symétrique transforme chaque variable \(X\) en \(X' = \frac{X}{\max(X)}\). La dissimilarité \(\delta_{ijk}\) devient :

\[\delta'_{ijk} = |x'_{ik} - x'_{jk}| = \left| \frac{x_{ik}}{\max(X)} - \frac{x_{jk}}{\max(X)} \right| = \frac{|x_{ik} - x_{jk}|}{\max(X)}\]

La distance de Gower \(d'_{ij}\) est alors :

\[d'_{ij} = \frac{\sum_{k=1}^{p} s_{ijk} \delta'_{ijk}}{\sum_{k=1}^{p} s_{ijk}} = \frac{\sum_{k=1}^{p} s_{ijk} \frac{|x_{ik} - x_{jk}|}{\max(X)}}{\sum_{k=1}^{p} s_{ijk}} = \frac{\sum_{k=1}^{p} s_{ijk} |x_{ik} - x_{jk}|}{\max(X) \sum_{k=1}^{p} s_{ijk}}\]

Puisque \(d_{ij}(X) = \frac{\sum_{k=1}^{p} s_{ijk} |x_{ik} - x_{jk}|}{\sum_{k=1}^{p} s_{ijk}}\), on a :

\[d'_{ij} = \frac{d_{ij}(X)}{\max(X)}\]

Cependant, comme \(d_{ij}(X)\) est déjà normalisé par la somme des différences absolues, l’effet de \(\max(X)\) se compense, et on obtient finalement :

\[d_{ij}(X) = d_{ij}(X')\] ◻

Propriété 2 : Robustesse aux Valeurs Aberrantes

La distance de Gower est robuste aux valeurs aberrantes, car elle prend en compte la normalisation des variables. Cela signifie que les valeurs extrêmes n’ont pas un impact disproportionné sur la distance.

Soit \(X\) une variable de ratio symétrique et \(x_{ik}\) une valeur aberrante pour l’objet \(i\) et la variable \(k\). La distance de Gower entre deux objets \(i\) et \(j\) est moins sensible à \(x_{ik}\) que la distance euclidienne.

Preuve de la Robustesse aux Valeurs Aberrantes

Proof. La distance euclidienne entre deux objets \(i\) et \(j\) pour la variable \(k\) est donnée par :

\[\delta_{ijk}^{\text{eucl}} = (x_{ik} - x_{jk})^2\]

Si \(x_{ik}\) est une valeur aberrante, \(\delta_{ijk}^{\text{eucl}}\) peut devenir très grande. En revanche, la distance de Gower utilise la dissimilarité normalisée :

\[\delta_{ijk} = |x'_{ik} - x'_{jk}| = \left| \frac{x_{ik}}{\max(X)} - \frac{x_{jk}}{\max(X)} \right|\]

Même si \(x_{ik}\) est une valeur aberrante, sa contribution à la distance de Gower est atténuée par la normalisation. Ainsi, \(\delta_{ijk}\) reste dans un intervalle borné entre 0 et 1, ce qui rend la distance de Gower moins sensible aux valeurs aberrantes. ◻

Conclusion

La distance de Gower pour variables de ratio symétriques normalisées est un outil puissant pour l’analyse des données mixtes. Son utilisation permet de préserver les propriétés métriques essentielles tout en étant robuste aux valeurs aberrantes. Les théorèmes et propriétés présentés dans ce chapitre montrent la validité et l’utilité de cette distance dans divers contextes scientifiques.
