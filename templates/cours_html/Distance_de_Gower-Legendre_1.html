Distance de Gower-Legendre : Une Mesure Multidimensionnelle pour l’Analyse des Données

Introduction et Motivations

L’analyse des données multidimensionnelles est une tâche complexe qui nécessite des outils adaptés pour capturer les relations entre les objets étudiés. Parmi ces outils, la distance de Gower-Legendre émerge comme une solution élégante et puissante pour mesurer la dissimilarité entre des objets décrits par des variables de différents types. Cette distance, introduite par les statisticiens Gower et Legendre, combine habilement des mesures de dissimilarité pour des variables continues, binaires et catégorielles, offrant ainsi une approche unifiée pour l’analyse des données mixtes.

L’origine de cette distance remonte aux travaux de Gower (1971) sur les méthodes d’analyse des données multidimensionnelles, suivis par les contributions de Legendre (1983) sur l’analyse des tableaux écologiques. La nécessité d’une telle mesure est évidente dans de nombreux domaines, tels que la biologie, la psychologie et les sciences sociales, où les données sont souvent décrites par des variables de types différents. La distance de Gower-Legendre permet de surmonter les limitations des distances traditionnelles, telles que la distance euclidienne, qui ne sont applicables qu’aux variables continues.

Dans ce chapitre, nous explorerons les fondements de la distance de Gower-Legendre, ses définitions formelles, ses propriétés mathématiques et ses applications pratiques. Nous montrerons comment cette distance peut être utilisée pour effectuer des analyses de clustering, des régressions et des tests d’hypothèses sur des données mixtes.

Définitions

Pour définir la distance de Gower-Legendre, nous devons d’abord comprendre ce que nous cherchons à mesurer. Imaginons que nous avons un ensemble d’objets décrits par plusieurs variables, certaines continues, d’autres binaires ou catégorielles. Nous voulons mesurer la dissimilarité entre deux objets en tenant compte de toutes ces variables. La distance de Gower-Legendre est une mesure qui combine les dissimilarités pour chaque type de variable.

Formellement, soit \(X\) une matrice de données de dimension \(n \times p\), où \(n\) est le nombre d’objets et \(p\) est le nombre de variables. Chaque variable peut être de type continu, binaire ou catégoriel. Pour chaque paire d’objets \(i\) et \(j\), la distance de Gower-Legendre est définie comme suit :

\[d_{ij} = \frac{\sum_{k=1}^{p} w_{ijk} d_{ijk}}{\sum_{k=1}^{p} w_{ijk}}\]

où \(d_{ijk}\) est la dissimilarité entre les objets \(i\) et \(j\) pour la variable \(k\), et \(w_{ijk}\) est un poids associé à cette dissimilarité. Les poids peuvent être utilisés pour donner plus d’importance à certaines variables.

Pour les variables continues, la dissimilarité est définie comme la différence absolue normalisée :

\[d_{ijk} = \frac{|x_{ik} - x_{jk}|}{max_k(x_k) - min_k(x_k)}\]

Pour les variables binaires, la dissimilarité est définie comme :

\[d_{ijk} = \begin{cases} 0 & \text{si } x_{ik} = x_{jk} \\ 1 & \text{sinon} \end{cases}\]

Pour les variables catégorielles, la dissimilarité est définie comme :

\[d_{ijk} = \begin{cases} 0 & \text{si } x_{ik} = x_{jk} \\ 1 & \text{sinon} \end{cases}\]

Théorèmes

Un théorème important lié à la distance de Gower-Legendre est le théorème de métrique, qui garantit que cette distance satisfait les propriétés d’une métrique. Formellement, nous avons le théorème suivant :

Soit \(d\) la distance de Gower-Legendre définie sur un ensemble d’objets \(X\). Alors, pour tout triplet d’objets \(i, j, k\), les propriétés suivantes sont satisfaites :

  1. \(d_{ij} \geq 0\) (non-négativité)

  2. \(d_{ij} = 0\) si et seulement si \(i = j\) (identité des indiscernables)

  3. \(d_{ij} = d_{ji}\) (symétrie)

  4. \(d_{ik} \leq d_{ij} + d_{jk}\) (inégalité triangulaire)

La démonstration de ce théorème repose sur les propriétés des dissimilarités pour chaque type de variable et sur la manière dont elles sont combinées. Nous allons maintenant détailler cette démonstration.

Preuves

Pour démontrer le théorème de métrique pour la distance de Gower-Legendre, nous devons montrer que chaque propriété est satisfaite.

Proof.

  1. Non-négativité : Pour toute paire d’objets \(i\) et \(j\), chaque dissimilarité \(d_{ijk}\) est non-négative, et les poids \(w_{ijk}\) sont également non-négatifs. Par conséquent, la somme des dissimilarités pondérées est non-négative.

    \[d_{ij} = \frac{\sum_{k=1}^{p} w_{ijk} d_{ijk}}{\sum_{k=1}^{p} w_{ijk}} \geq 0\]

  2. Identité des indiscernables : Si \(i = j\), alors pour toute variable \(k\), \(d_{ijk} = 0\). Par conséquent, la distance \(d_{ij}\) est nulle.

    \[d_{ij} = \frac{\sum_{k=1}^{p} w_{ijk} \cdot 0}{\sum_{k=1}^{p} w_{ijk}} = 0\]

    Réciproquement, si \(d_{ij} = 0\), alors toutes les dissimilarités \(d_{ijk}\) doivent être nulles, ce qui implique que \(i = j\).

  3. Symétrie : La distance de Gower-Legendre est symétrique car les dissimilarités \(d_{ijk}\) sont symétriques pour chaque type de variable.

    \[d_{ij} = \frac{\sum_{k=1}^{p} w_{ijk} d_{ijk}}{\sum_{k=1}^{p} w_{ijk}} = \frac{\sum_{k=1}^{p} w_{jik} d_{jik}}{\sum_{k=1}^{p} w_{jik}} = d_{ji}\]

  4. Inégalité triangulaire : Pour montrer l’inégalité triangulaire, nous utilisons le fait que chaque dissimilarité \(d_{ijk}\) satisfait l’inégalité triangulaire. Par conséquent, la distance de Gower-Legendre hérite de cette propriété.

    \[d_{ik} \leq d_{ij} + d_{jk}\]

    Cette propriété découle directement des propriétés des dissimilarités pour chaque type de variable et de la manière dont elles sont combinées.

Propriétés et Corollaires

La distance de Gower-Legendre possède plusieurs propriétés intéressantes qui en font un outil puissant pour l’analyse des données. Nous allons maintenant les énumérer et les démontrer.

  1. Normalisation : La distance de Gower-Legendre est normalisée, c’est-à-dire que pour toute paire d’objets \(i\) et \(j\), la distance est comprise entre 0 et 1.

    \[0 \leq d_{ij} \leq 1\]

    Cette propriété découle du fait que chaque dissimilarité \(d_{ijk}\) est normalisée entre 0 et 1.

  2. Invariance par transformation linéaire : La distance de Gower-Legendre est invariante par transformation linéaire des variables continues. Cela signifie que si nous appliquons une transformation affine à une variable continue, la distance entre deux objets reste inchangée.

    \[d_{ij} = \frac{\sum_{k=1}^{p} w_{ijk} d_{ijk}}{\sum_{k=1}^{p} w_{ijk}} = \frac{\sum_{k=1}^{p} w_{ijk} d'_{ijk}}{\sum_{k=1}^{p} w_{ijk}} = d'_{ij}\]

    où \(d'_{ijk}\) est la dissimilarité pour la variable transformée.

  3. Consistance avec les types de variables : La distance de Gower-Legendre est conçue pour être cohérente avec les types de variables. Cela signifie que la dissimilarité pour chaque type de variable est définie de manière appropriée, en tenant compte des caractéristiques spécifiques du type de variable.

Conclusion

La distance de Gower-Legendre est un outil puissant et flexible pour mesurer la dissimilarité entre des objets décrits par des variables de différents types. Ses propriétés mathématiques et ses applications pratiques en font un choix privilégié pour l’analyse des données multidimensionnelles. Dans ce chapitre, nous avons exploré les fondements de cette distance, ses définitions formelles, ses propriétés et ses applications. Nous avons également démontré le théorème de métrique pour cette distance, montrant ainsi qu’elle satisfait les propriétés d’une métrique.

Les perspectives d’application de la distance de Gower-Legendre sont vastes et variées. Elle peut être utilisée dans de nombreux domaines, tels que la biologie, la psychologie et les sciences sociales, pour effectuer des analyses de clustering, des régressions et des tests d’hypothèses sur des données mixtes. En outre, cette distance peut être étendue et adaptée pour répondre à des besoins spécifiques, ouvrant ainsi de nouvelles voies de recherche et d’application.
