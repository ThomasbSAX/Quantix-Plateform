L’inégalité de Minkowski pour les séries

Quantix

2026-01-11

Introduction et Motivations

L’inégalité de Minkowski pour les séries est une généralisation fondamentale de l’inégalité de Minkowski classique, qui elle-même trouve ses racines dans les travaux de Hermann Minkowski sur la géométrie des nombres. Cette inégalité est indispensable dans l’étude des espaces \(L^p\), où elle permet de contrôler la norme de la somme de deux fonctions intégrables. Lorsque l’on considère des séries, cette inégalité devient un outil puissant pour établir la convergence et estimer les sommes partielles.

L’émergence de cette notion est motivée par le besoin de généraliser les résultats classiques de l’analyse à des espaces fonctionnels plus complexes. Elle est particulièrement utile dans le cadre de la théorie des séries et des intégrales, où elle permet de passer d’une somme ou d’une intégrale finie à une somme ou une intégrale infinie de manière contrôlée.

Définitions

Avant d’énoncer l’inégalité de Minkowski pour les séries, il est essentiel de rappeler quelques notions fondamentales.

La norme \(L^p\)

Considérons un espace mesuré \((X, \mathcal{A}, \mu)\). Pour \(1 \leq p < \infty\), la norme \(L^p\) d’une fonction mesurable \(f : X \to \mathbb{R}\) est définie comme suit :

La norme \(L^p\) de \(f\) est donnée par : \[\| f \|_{L^p} = \left( \int_X |f(x)|^p \, d\mu(x) \right)^{1/p}\]

Pour \(p = \infty\), la norme \(L^\infty\) est définie par : \[\| f \|_{L^\infty} = \inf \{ M \geq 0 : |f(x)| \leq M \text{ presque partout} \}\]

Séries dans \(L^p\)

Considérons une suite de fonctions \((f_n)_{n \in \mathbb{N}}\) dans \(L^p(X, \mathcal{A}, \mu)\). La série \(\sum_{n=0}^\infty f_n\) est dite convergente dans \(L^p\) si la suite des sommes partielles \((\sum_{n=0}^N f_n)_{N \in \mathbb{N}}\) converge dans \(L^p\).

Inégalité de Minkowski pour les séries

Nous allons maintenant énoncer et démontrer l’inégalité de Minkowski pour les séries.

Énoncé

Soit \((X, \mathcal{A}, \mu)\) un espace mesuré et \(1 \leq p < \infty\). Considérons deux suites de fonctions \((f_n)_{n \in \mathbb{N}}\) et \((g_n)_{n \in \mathbb{N}}\) dans \(L^p(X, \mathcal{A}, \mu)\) telles que les séries \(\sum_{n=0}^\infty f_n\) et \(\sum_{n=0}^\infty g_n\) convergent dans \(L^p\). Alors, pour tout \(N \in \mathbb{N}\), on a : \[\left\| \sum_{n=0}^N (f_n + g_n) \right\|_{L^p} \leq \left\| \sum_{n=0}^N f_n \right\|_{L^p} + \left\| \sum_{n=0}^N g_n \right\|_{L^p}\]

Démonstration

Nous allons démontrer cette inégalité en plusieurs étapes.

Proof. Soit \(N \in \mathbb{N}\). Considérons les sommes partielles : \[S_N = \sum_{n=0}^N f_n, \quad T_N = \sum_{n=0}^N g_n\] Nous voulons montrer que : \[\| S_N + T_N \|_{L^p} \leq \| S_N \|_{L^p} + \| T_N \|_{L^p}\]

Pour cela, nous utilisons l’inégalité de Minkowski classique pour les intégrales. Appliquons cette inégalité à la fonction \(S_N + T_N\) : \[\| S_N + T_N \|_{L^p} = \left( \int_X |S_N(x) + T_N(x)|^p \, d\mu(x) \right)^{1/p}\]

En utilisant l’inégalité de Minkowski pour les intégrales, nous avons : \[\left( \int_X |S_N(x) + T_N(x)|^p \, d\mu(x) \right)^{1/p} \leq \int_X |S_N(x) + T_N(x)| \, d\mu(x)\]

De plus, par l’inégalité triangulaire pour les fonctions, nous avons : \[\int_X |S_N(x) + T_N(x)| \, d\mu(x) \leq \int_X |S_N(x)| + |T_N(x)| \, d\mu(x)\]

En appliquant à nouveau l’inégalité de Minkowski pour les intégrales, nous obtenons : \[\int_X |S_N(x)| + |T_N(x)| \, d\mu(x) \leq \left( \int_X |S_N(x)|^p \, d\mu(x) \right)^{1/p} + \left( \int_X |T_N(x)|^p \, d\mu(x) \right)^{1/p}\]

Finalement, nous avons : \[\| S_N + T_N \|_{L^p} \leq \| S_N \|_{L^p} + \| T_N \|_{L^p}\]

Ceci achève la démonstration. ◻

Propriétés et Corollaires

Nous allons maintenant énoncer quelques propriétés et corollaires de l’inégalité de Minkowski pour les séries.

Propriétés

  1. L’inégalité de Minkowski pour les séries est valable pour tout \(1 \leq p < \infty\). Pour \(p = \infty\), l’inégalité devient une égalité : \[\| S_N + T_N \|_{L^\infty} = \max(\| S_N \|_{L^\infty}, \| T_N \|_{L^\infty})\]

  2. L’inégalité de Minkowski pour les séries est une généralisation de l’inégalité triangulaire dans \(L^p\). En effet, si nous prenons \(N = 0\), nous retrouvons l’inégalité triangulaire classique : \[\| f_0 + g_0 \|_{L^p} \leq \| f_0 \|_{L^p} + \| g_0 \|_{L^p}\]

  3. L’inégalité de Minkowski pour les séries permet de contrôler la convergence des séries dans \(L^p\). En particulier, si les suites \((f_n)_{n \in \mathbb{N}}\) et \((g_n)_{n \in \mathbb{N}}\) sont telles que les séries \(\sum_{n=0}^\infty f_n\) et \(\sum_{n=0}^\infty g_n\) convergent dans \(L^p\), alors la série \(\sum_{n=0}^\infty (f_n + g_n)\) converge également dans \(L^p\).

Corollaires

Soit \((X, \mathcal{A}, \mu)\) un espace mesuré et \(1 \leq p < \infty\). Considérons une suite de fonctions \((f_n)_{n \in \mathbb{N}}\) dans \(L^p(X, \mathcal{A}, \mu)\) telle que la série \(\sum_{n=0}^\infty f_n\) converge dans \(L^p\). Alors, pour tout \(N \in \mathbb{N}\), on a : \[\left\| \sum_{n=0}^N f_n \right\|_{L^p} \leq \sum_{n=0}^N \| f_n \|_{L^p}\]

Proof. Nous utilisons l’inégalité de Minkowski pour les séries en prenant \(g_n = 0\) pour tout \(n \in \mathbb{N}\). Nous avons : \[\left\| \sum_{n=0}^N f_n \right\|_{L^p} = \left\| \sum_{n=0}^N (f_n + 0) \right\|_{L^p} \leq \left\| \sum_{n=0}^N f_n \right\|_{L^p} + \left\| \sum_{n=0}^N 0 \right\|_{L^p} = \left\| \sum_{n=0}^N f_n \right\|_{L^p}\]

De plus, en utilisant l’inégalité de Minkowski pour les intégrales, nous avons : \[\left\| \sum_{n=0}^N f_n \right\|_{L^p} \leq \sum_{n=0}^N \| f_n \|_{L^p}\]

Ceci achève la démonstration. ◻

Conclusion

L’inégalité de Minkowski pour les séries est un outil fondamental dans l’étude des espaces \(L^p\) et des séries de fonctions. Elle permet de contrôler la convergence des séries et d’établir des inégalités importantes pour les sommes partielles. Cette inégalité trouve ses applications dans de nombreux domaines de l’analyse, notamment dans la théorie des intégrales et des séries.
