Coefficients multinomiaux : Fondements et Applications

Quantix

2026-01-11

Introduction et Motivations

Les coefficients multinomiaux émergent naturellement dans l’étude des généralisations du binôme de Newton. Alors que les coefficients binomiaux \(\binom{n}{k}\) comptent le nombre de combinaisons de \(k\) éléments parmi \(n\), les coefficients multinomiaux \(\binom{n}{k_1, k_2, \ldots, k_m}\) étendent cette notion en considérant des partitions de \(n\) éléments en \(m\) sous-ensembles distincts.

Historiquement, ces coefficients apparaissent dans les travaux de Blaise Pascal et Abraham de Moivre au XVIIe siècle. Leur importance réside dans leur capacité à modéliser des phénomènes combinatoires complexes, tels que la distribution de particules dans plusieurs compartiments ou le développement de polynômes multivariés.

Définitions

Pour introduire les coefficients multinomiaux, considérons un ensemble de \(n\) éléments que nous souhaitons partitionner en \(m\) sous-ensembles distincts, de tailles respectives \(k_1, k_2, \ldots, k_m\) telles que \(\sum_{i=1}^m k_i = n\).

Le coefficient multinomial \(\binom{n}{k_1, k_2, \ldots, k_m}\) est défini comme le nombre de façons de partitionner \(n\) éléments en \(m\) sous-ensembles distincts, avec exactement \(k_i\) éléments dans le \(i\)-ème sous-ensemble.

Formellement, nous avons :

\[\binom{n}{k_1, k_2, \ldots, k_m} = \frac{n!}{k_1! k_2! \cdots k_m!}\]

où \(n!\) désigne la factorielle de \(n\), c’est-à-dire le produit de tous les entiers positifs jusqu’à \(n\).

Théorèmes

Un théorème fondamental concernant les coefficients multinomiaux est le suivant :

Soient \(n\) un entier positif et \(k_1, k_2, \ldots, k_m\) des entiers positifs tels que \(\sum_{i=1}^m k_i = n\). Alors, le coefficient multinomial satisfait la relation suivante :

\[\binom{n}{k_1, k_2, \ldots, k_m} = \frac{n!}{k_1! k_2! \cdots k_m!}\]

De plus, pour tout polynôme \(P(x_1, x_2, \ldots, x_m)\) de degré \(n\), le développement multinomial est donné par :

\[P(x_1, x_2, \ldots, x_m)^n = \sum_{\substack{k_1 + k_2 + \cdots + k_m = n \\ k_i \geq 0}} \binom{n}{k_1, k_2, \ldots, k_m} x_1^{k_1} x_2^{k_2} \cdots x_m^{k_m}\]

Preuves

Pour démontrer le théorème de généralisation du binôme, nous procédons par induction sur \(n\).

Proof. Base de l’induction : Pour \(n = 1\), nous avons un seul élément à partitionner en \(m\) sous-ensembles. Il y a exactement \(m\) façons de placer cet élément dans l’un des sous-ensembles. Ainsi,

\[\binom{1}{k_1, k_2, \ldots, k_m} = m\]

ce qui correspond bien à \(\frac{1!}{k_1! k_2! \cdots k_m!}\) lorsque \(\sum_{i=1}^m k_i = 1\).

Hypothèse d’induction : Supposons que le théorème est vrai pour un certain entier \(n \geq 1\), c’est-à-dire que

\[\binom{n}{k_1, k_2, \ldots, k_m} = \frac{n!}{k_1! k_2! \cdots k_m!}\]

pour tout \(k_1, k_2, \ldots, k_m\) tels que \(\sum_{i=1}^m k_i = n\).

Étape d’induction : Considérons \(n + 1\). Nous devons montrer que

\[\binom{n+1}{k_1, k_2, \ldots, k_m} = \frac{(n+1)!}{k_1! k_2! \cdots k_m!}\]

pour tout \(k_1, k_2, \ldots, k_m\) tels que \(\sum_{i=1}^m k_i = n + 1\).

Par définition, le coefficient multinomial \(\binom{n+1}{k_1, k_2, \ldots, k_m}\) compte le nombre de façons de partitionner \(n+1\) éléments en \(m\) sous-ensembles distincts, avec exactement \(k_i\) éléments dans le \(i\)-ème sous-ensemble.

Nous pouvons fixer un élément parmi les \(n+1\) et le placer dans l’un des \(m\) sous-ensembles. Supposons que nous plaçons cet élément dans le premier sous-ensemble. Alors, il reste \(n\) éléments à partitionner en \(m\) sous-ensembles, avec exactement \(k_1 - 1\) éléments dans le premier sous-ensemble et \(k_i\) éléments dans les autres sous-ensembles pour \(i \geq 2\).

Par l’hypothèse d’induction, le nombre de façons de partitionner ces \(n\) éléments est

\[\binom{n}{k_1 - 1, k_2, \ldots, k_m} = \frac{n!}{(k_1 - 1)! k_2! \cdots k_m!}\]

Puisque nous avons \(m\) choix pour placer l’élément fixé, le nombre total de partitions est

\[\binom{n+1}{k_1, k_2, \ldots, k_m} = m \cdot \frac{n!}{(k_1 - 1)! k_2! \cdots k_m!} = \frac{(n+1)!}{k_1! k_2! \cdots k_m!}\]

ce qui achève la démonstration par induction. ◻

Propriétés et Corollaires

Les coefficients multinomiaux possèdent plusieurs propriétés intéressantes :

  1. Symétrie : Pour tout permutation \(\sigma\) de \(\{1, 2, \ldots, m\}\), nous avons

    \[\binom{n}{k_{\sigma(1)}, k_{\sigma(2)}, \ldots, k_{\sigma(m)}} = \binom{n}{k_1, k_2, \ldots, k_m}\]

    Cette propriété découle du fait que les sous-ensembles sont indistinguables à l’exception de leur taille.

  2. Relation de récurrence : Pour tout \(n, k_1, k_2, \ldots, k_m\) tels que \(\sum_{i=1}^m k_i = n\), nous avons

    \[\binom{n}{k_1, k_2, \ldots, k_m} = \sum_{j=0}^k \binom{n-1}{j, k_2, \ldots, k_m}\]

    où la somme est prise sur tous les \(j\) tels que \(\sum_{i=2}^m k_i = n - j\).

  3. Inégalité : Pour tout \(n, k_1, k_2, \ldots, k_m\) tels que \(\sum_{i=1}^m k_i = n\), nous avons

    \[\binom{n}{k_1, k_2, \ldots, k_m} \leq \frac{n^m}{k_1! k_2! \cdots k_m!}\]

    Cette inégalité est une conséquence directe de la définition des coefficients multinomiaux et du principe de comptage.

Conclusion

Les coefficients multinomiaux constituent un outil puissant en combinatoire et en algèbre. Leur généralisation des coefficients binomiaux permet de modéliser une large gamme de problèmes, allant de la distribution de particules à l’analyse des polynômes multivariés. Les propriétés et théorèmes associés offrent une richesse mathématique qui continue d’inspirer des recherches dans divers domaines des mathématiques.
