La distance de Gower pour variables d’intervalle asymétriques normalisées

Quantix

2026-01-11

Introduction et Motivations

L’analyse des données multidimensionnelles est un pilier fondamental de la statistique moderne. Parmi les défis rencontrés, l’évaluation des distances entre observations prend une place prépondérante. La distance de Gower, introduite par Gower en 1971, offre une solution élégante pour mesurer la dissimilarité entre objets décrits par des variables de différents types. Cependant, son application aux variables d’intervalle asymétriques normalisées reste peu explorée.

Les variables d’intervalle, définies sur un domaine borné mais non nécessairement symétrique, sont omniprésentes en sciences sociales et économiques. Leur normalisation asymétrique permet de préserver l’information tout en standardisant les échelles. La distance de Gower, par sa capacité à intégrer divers types de variables, se révèle alors particulièrement adaptée.

Ce chapitre explore la généralisation de la distance de Gower aux variables d’intervalle asymétriques normalisées. Nous commencerons par définir rigoureusement ces concepts, avant d’établir les propriétés mathématiques de cette nouvelle distance. Des exemples concrets illustreront son utilité dans l’analyse des données.

Définitions

Variables d’intervalle asymétriques normalisées

Considérons un ensemble de variables \(X_1, \ldots, X_p\) définies sur des intervalles bornés. Chaque variable \(X_j\) prend ses valeurs dans un intervalle \([a_j, b_j]\), où \(a_j\) et \(b_j\) sont les bornes inférieures et supérieures respectives.

Pour normaliser ces variables de manière asymétrique, nous introduisons une transformation qui préserve la structure asymétrique des intervalles. Intuitivement, cette normalisation doit permettre de comparer les observations sur une échelle commune tout en tenant compte des différences d’échelle et de position des intervalles.

Soit \(X_j\) une variable définie sur \([a_j, b_j]\). La normalisation asymétrique de \(X_j\) est définie par la transformation : \[T_j: [a_j, b_j] \rightarrow [0, 1], \quad x \mapsto T_j(x) = \frac{x - a_j}{b_j - a_j}\]

Cette transformation linéaire préserve l’ordre des valeurs tout en normalisant l’intervalle sur \([0, 1]\). La distance entre deux observations \(x\) et \(y\) pour la variable \(X_j\) peut alors être définie comme : \[d_j(x, y) = |T_j(x) - T_j(y)| = \left| \frac{x - a_j}{b_j - a_j} - \frac{y - a_j}{b_j - a_j} \right| = \frac{|x - y|}{b_j - a_j}\]

Distance de Gower

La distance de Gower est une mesure de dissimilarité qui combine plusieurs types de variables. Pour des variables d’intervalle, elle est définie comme la somme pondérée des distances normalisées.

Intuitivement, cette distance doit permettre de comparer deux observations en tenant compte de toutes les variables disponibles, tout en normalisant l’influence de chaque variable.

Soit \(\mathbf{X} = (X_1, \ldots, X_p)\) un ensemble de variables d’intervalle. La distance de Gower entre deux observations \(\mathbf{x} = (x_1, \ldots, x_p)\) et \(\mathbf{y} = (y_1, \ldots, y_p)\) est définie par : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j d_j(x_j, y_j)\] où \(w_j\) est le poids associé à la variable \(X_j\), et \(d_j(x_j, y_j)\) est la distance normalisée pour la variable \(X_j\).

Pour des variables d’intervalle asymétriques normalisées, la distance \(d_j(x_j, y_j)\) est donnée par : \[d_j(x_j, y_j) = \frac{|x_j - y_j|}{b_j - a_j}\] Ainsi, la distance de Gower devient : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b_j - a_j}\]

Théorèmes

Propriétés de la distance de Gower

La distance de Gower doit satisfaire certaines propriétés fondamentales pour être une mesure de dissimilarité valide. Nous établissons ici ces propriétés pour les variables d’intervalle asymétriques normalisées.

La distance de Gower \(d_G\) définie par : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b_j - a_j}\] satisfait les propriétés suivantes :

  1. Non-négativité : \(d_G(\mathbf{x}, \mathbf{y}) \geq 0\) pour tout \(\mathbf{x}, \mathbf{y}\).

  2. Identité des indiscernables : \(d_G(\mathbf{x}, \mathbf{y}) = 0\) si et seulement si \(\mathbf{x} = \mathbf{y}\).

  3. Symétrie : \(d_G(\mathbf{x}, \mathbf{y}) = d_G(\mathbf{y}, \mathbf{x})\).

  4. Inégalité triangulaire : \(d_G(\mathbf{x}, \mathbf{z}) \leq d_G(\mathbf{x}, \mathbf{y}) + d_G(\mathbf{y}, \mathbf{z})\).

Démonstration des propriétés

Nous démontrons chacune de ces propriétés en détail.

Preuve des propriétés. 1. Non-négativité : Par définition, \(|x_j - y_j| \geq 0\) et \(b_j - a_j > 0\). Ainsi, chaque terme \(w_j \frac{|x_j - y_j|}{b_j - a_j}\) est non-négatif. La somme de termes non-négatifs est également non-négative.

2. Identité des indiscernables : \(d_G(\mathbf{x}, \mathbf{y}) = 0\) implique que chaque terme \(w_j \frac{|x_j - y_j|}{b_j - a_j} = 0\). Puisque \(w_j > 0\) et \(b_j - a_j > 0\), cela implique \(|x_j - y_j| = 0\), c’est-à-dire \(x_j = y_j\) pour tout \(j\). Réciproquement, si \(\mathbf{x} = \mathbf{y}\), alors \(d_G(\mathbf{x}, \mathbf{y}) = 0\).

3. Symétrie : Par la symétrie de la valeur absolue, \(|x_j - y_j| = |y_j - x_j|\). Ainsi, \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b_j - a_j} = \frac{1}{p} \sum_{j=1}^p w_j \frac{|y_j - x_j|}{b_j - a_j} = d_G(\mathbf{y}, \mathbf{x})\]

4. Inégalité triangulaire : Pour chaque variable \(X_j\), nous avons : \[\frac{|x_j - z_j|}{b_j - a_j} \leq \frac{|x_j - y_j|}{b_j - a_j} + \frac{|y_j - z_j|}{b_j - a_j}\] En multipliant par \(w_j\) et en sommant sur \(j\), nous obtenons : \[\sum_{j=1}^p w_j \frac{|x_j - z_j|}{b_j - a_j} \leq \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b_j - a_j} + \sum_{j=1}^p w_j \frac{|y_j - z_j|}{b_j - a_j}\] En divisant par \(p\), nous avons : \[d_G(\mathbf{x}, \mathbf{z}) \leq d_G(\mathbf{x}, \mathbf{y}) + d_G(\mathbf{y}, \mathbf{z})\] ◻

Propriétés et Corollaires

Propriétés supplémentaires

Nous listons ici quelques propriétés supplémentaires de la distance de Gower pour les variables d’intervalle asymétriques normalisées.

La distance de Gower \(d_G\) satisfait les propriétés suivantes :

  1. Si toutes les variables \(X_j\) ont le même intervalle \([a, b]\), alors la distance de Gower se simplifie en : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j |x_j - y_j|\]

  2. Si toutes les variables \(X_j\) ont le même poids \(w_j = 1\), alors la distance de Gower est la moyenne des distances normalisées : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p \frac{|x_j - y_j|}{b_j - a_j}\]

  3. La distance de Gower est invariante par translation et homothétie des variables. Plus précisément, si \(X_j' = c_j + d_j X_j\) avec \(d_j > 0\), alors la distance de Gower entre \(\mathbf{x}'\) et \(\mathbf{y}'\) est égale à la distance de Gower entre \(\mathbf{x}\) et \(\mathbf{y}\).

Démonstration des propriétés supplémentaires

Nous démontrons chacune de ces propriétés en détail.

Preuve des propriétés supplémentaires. (i) Si toutes les variables \(X_j\) ont le même intervalle \([a, b]\), alors \(b_j - a_j = b - a\) pour tout \(j\). Ainsi, \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b - a} = \frac{1}{p(b - a)} \sum_{j=1}^p w_j |x_j - y_j|\] Cependant, si nous choisissons \(w_j = b - a\) pour tout \(j\), alors la distance de Gower se simplifie en : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p |x_j - y_j|\]

(ii) Si toutes les variables \(X_j\) ont le même poids \(w_j = 1\), alors la distance de Gower est simplement la moyenne des distances normalisées : \[d_G(\mathbf{x}, \mathbf{y}) = \frac{1}{p} \sum_{j=1}^p \frac{|x_j - y_j|}{b_j - a_j}\]

(iii) Considérons une transformation linéaire \(X_j' = c_j + d_j X_j\) avec \(d_j > 0\). La distance de Gower entre \(\mathbf{x}'\) et \(\mathbf{y}'\) est : \[d_G(\mathbf{x}', \mathbf{y}') = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j' - y_j'|}{b_j' - a_j'}\] En utilisant \(x_j' = c_j + d_j x_j\) et \(y_j' = c_j + d_j y_j\), nous avons : \[|x_j' - y_j'| = d_j |x_j - y_j|\] De plus, \(b_j' - a_j' = d_j (b_j - a_j)\). Ainsi, \[d_G(\mathbf{x}', \mathbf{y}') = \frac{1}{p} \sum_{j=1}^p w_j \frac{d_j |x_j - y_j|}{d_j (b_j - a_j)} = \frac{1}{p} \sum_{j=1}^p w_j \frac{|x_j - y_j|}{b_j - a_j} = d_G(\mathbf{x}, \mathbf{y})\] ◻

Conclusion

La distance de Gower pour variables d’intervalle asymétriques normalisées offre une solution robuste et flexible pour l’analyse des données multidimensionnelles. En intégrant les propriétés de normalisation asymétrique, cette distance permet de comparer des observations sur des échelles différentes tout en préservant l’information contenue dans les intervalles.

Les propriétés mathématiques établies dans ce chapitre garantissent que la distance de Gower est une mesure valide de dissimilarité. Les applications potentielles sont nombreuses, allant de l’analyse des données économiques aux études sociales.

Pour les recherches futures, il serait intéressant d’explorer des extensions de cette distance à d’autres types de variables ou à des contextes plus complexes. L’intégration de méthodes d’apprentissage automatique pourrait également enrichir les applications pratiques de cette distance.
