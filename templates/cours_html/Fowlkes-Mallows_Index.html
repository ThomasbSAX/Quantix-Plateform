Fowlkes-Mallows Index: A Measure of Cluster Similarity

2026-01-11

Introduction and Motivations

The Fowlkes-Mallows Index (FMI) is a metric used to evaluate the similarity between two clusterings of a dataset. It was introduced by Fowlkes and Mallows in 1983 as an alternative to the Rand Index, providing a more interpretable measure of clustering similarity. The FMI is particularly useful in the context of comparing the results of different clustering algorithms or evaluating the stability of a clustering method.

The need for such an index arises from the fact that clustering is an unsupervised learning task, meaning there is no ground truth to compare against. Instead, we often want to compare different clustering results to understand their similarities and differences.

Definitions

To understand the Fowlkes-Mallows Index, we first need to define some basic concepts.

A clustering of a dataset \(X = \{x_1, x_2, \dots, x_n\}\) is a partition of \(X\) into \(k\) subsets \(C = \{C_1, C_2, \dots, C_k\}\) such that: \[\bigcup_{i=1}^k C_i = X \quad \text{and} \quad C_i \cap C_j = \emptyset \quad \forall i \neq j.\]

Given two clusterings \(C\) and \(C'\) of the same dataset, we can define a contingency table where each entry \(n_{ij}\) represents the number of points that are in cluster \(C_i\) and cluster \(C'_j\).

For two clusterings \(C = \{C_1, \dots, C_k\}\) and \(C' = \{C'_1, \dots, C'_l\}\), the contingency table \(n\) is defined as: \[n_{ij} = |C_i \cap C'_j|, \quad \forall i \in \{1, \dots, k\}, j \in \{1, \dots, l\}.\]

The Fowlkes-Mallows Index is then defined based on the precision and recall of the clusters.

Fowlkes-Mallows Index

Given two clusterings \(C\) and \(C'\), the Fowlkes-Mallows Index is defined as: \[\text{FMI}(C, C') = \sqrt{\frac{TP}{FP} \cdot \frac{TP}{FN}},\] where \(TP\) (true positives), \(FP\) (false positives), and \(FN\) (false negatives) are defined based on the contingency table.

To compute \(TP\), \(FP\), and \(FN\), we need to consider pairs of points. For each pair of points, we can define:

  • \(TP\): The number of pairs of points that are in the same cluster in both \(C\) and \(C'\).

  • \(FP\): The number of pairs of points that are in the same cluster in \(C\) but not in \(C'\).

  • \(FN\): The number of pairs of points that are in the same cluster in \(C'\) but not in \(C\).

These can be computed as follows: \[TP = \sum_{i=1}^k \sum_{j=1}^l \binom{n_{ij}}{2}, \quad FP = \sum_{i=1}^k \left( \binom{|C_i|}{2} - \sum_{j=1}^l \binom{n_{ij}}{2} \right), \quad FN = \sum_{j=1}^l \left( \binom{|C'_j|}{2} - \sum_{i=1}^k \binom{n_{ij}}{2} \right).\]

Properties

The Fowlkes-Mallows Index has several important properties:

  1. The FMI is symmetric, i.e., \(\text{FMI}(C, C') = \text{FMI}(C', C)\).

  2. The FMI ranges from 0 to 1, where 1 indicates perfect agreement between the two clusterings and 0 indicates no agreement.

  3. The FMI is not affected by the number of clusters, making it a robust measure for comparing clusterings with different numbers of clusters.

Consider a dataset with 4 points \(X = \{x_1, x_2, x_3, x_4\}\). Suppose we have two clusterings: \[C = \{ \{x_1, x_2\}, \{x_3, x_4\} \} \quad \text{and} \quad C' = \{ \{x_1, x_3\}, \{x_2, x_4\} \}.\]

The contingency table is: \[n = \begin{pmatrix} 1 & 1 \\ 1 & 1 \end{pmatrix},\] where \(n_{11} = |\{x_1, x_2\} \cap \{x_1, x_3\}| = 1\), \(n_{12} = |\{x_1, x_2\} \cap \{x_2, x_4\}| = 1\), and so on.

Computing \(TP\), \(FP\), and \(FN\): \[TP = \sum_{i=1}^2 \sum_{j=1}^2 \binom{n_{ij}}{2} = 0 + 0 + 0 + 0 = 0,\] \[FP = \sum_{i=1}^2 \left( \binom{|C_i|}{2} - \sum_{j=1}^2 \binom{n_{ij}}{2} \right) = 0 + 0 = 0,\] \[FN = \sum_{j=1}^2 \left( \binom{|C'_j|}{2} - \sum_{i=1}^2 \binom{n_{ij}}{2} \right) = 0 + 0 = 0.\]

Thus, the FMI is: \[\text{FMI}(C, C') = \sqrt{\frac{0}{0} \cdot \frac{0}{0}},\] which is undefined in this case. This example shows that the FMI can be undefined for certain clusterings.

Conclusion

The Fowlkes-Mallows Index is a valuable tool for comparing clusterings, providing a measure of similarity that is interpretable and robust. While it has some limitations, such as being undefined for certain clusterings, it remains a widely used metric in the field of clustering analysis.
