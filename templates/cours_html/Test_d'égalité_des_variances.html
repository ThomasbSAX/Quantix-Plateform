Test d’égalité des variances

Quantix

2026-01-11

Introduction et Motivations

L’étude des variances est un pilier fondamental dans l’analyse statistique. Le test d’égalité des variances, souvent réalisé via le test de Bartlett ou le test de Levene, permet de comparer les dispersions de plusieurs échantillons. Cette comparaison est cruciale dans de nombreux domaines, notamment en biologie pour l’analyse des variations génétiques, en économie pour évaluer la stabilité des marchés, ou encore en ingénierie pour assurer la cohérence des matériaux.

L’origine de ces tests remonte aux travaux pionniers de statisticiens tels que Bartlett et Levene. Le test de Bartlett, développé en 1937, est particulièrement puissant lorsque les données suivent une distribution normale. En revanche, le test de Levene, proposé en 1960, est plus robuste face aux violations de cette hypothèse. Ces tests sont indispensables pour valider l’application d’autres procédures statistiques, comme l’analyse de variance (ANOVA), qui suppose l’homogénéité des variances entre les groupes.

Définitions

Pour comprendre le test d’égalité des variances, il est essentiel de définir quelques concepts clés. Supposons que nous ayons \(k\) échantillons indépendants, chacun de taille \(n_i\), avec \(i = 1, 2, \ldots, k\). Nous notons \(X_{ij}\) la \(j\)-ième observation du \(i\)-ème échantillon.

L’objectif est de tester l’hypothèse nulle \(H_0 : \sigma_1^2 = \sigma_2^2 = \ldots = \sigma_k^2\), où \(\sigma_i^2\) représente la variance de l’échantillon \(i\). Cette hypothèse est testée contre une hypothèse alternative \(H_1\) qui affirme que les variances ne sont pas toutes égales.

Test de Bartlett

Le test de Bartlett est basé sur le logarithme des estimations des variances. La statistique de test \(B\) est définie comme suit :

\[B = \frac{(N - k) \ln(s_p^2) - \sum_{i=1}^k (n_i - 1) \ln(s_i^2)}{C}\]

où : - \(N = \sum_{i=1}^k n_i\) est la taille totale de l’échantillon, - \(s_p^2 = \frac{\sum_{i=1}^k (n_i - 1) s_i^2}{N - k}\) est la variance pondérée, - \(s_i^2 = \frac{1}{n_i - 1} \sum_{j=1}^{n_i} (X_{ij} - \bar{X}_i)^2\) est la variance de l’échantillon \(i\), - \(C = 1 + \frac{1}{3(k - 1)} \left( \sum_{i=1}^k \frac{1}{n_i - 1} - \frac{1}{N - k} \right)\).

Test de Levene

Le test de Levene est basé sur les résidus absolutisés. La statistique de test \(W\) est définie comme suit :

\[W = \frac{(N - k) \sum_{i=1}^k n_i (\bar{Z}_{i..} - \bar{Z}_{...})^2}{(k - 1) \sum_{i=1}^k (n_i - 1) s_{Z_i}^2}\]

où : - \(Z_{ij} = |X_{ij} - \bar{X}_i|\) est le résidu absolutisé, - \(\bar{Z}_{i..} = \frac{1}{n_i} \sum_{j=1}^{n_i} Z_{ij}\) est la moyenne des résidus absolutisés pour l’échantillon \(i\), - \(\bar{Z}_{...} = \frac{1}{N} \sum_{i=1}^k \sum_{j=1}^{n_i} Z_{ij}\) est la moyenne globale des résidus absolutisés, - \(s_{Z_i}^2 = \frac{1}{n_i - 1} \sum_{j=1}^{n_i} (Z_{ij} - \bar{Z}_{i..})^2\) est la variance des résidus absolutisés pour l’échantillon \(i\).

Théorèmes

Théorème de Bartlett

Sous l’hypothèse nulle \(H_0\), la statistique de test \(B\) suit une distribution du chi-deux à \(k - 1\) degrés de liberté lorsque les échantillons sont indépendants et normalement distribués.

\[B \sim \chi^2_{k - 1}\]

Théorème de Levene

Sous l’hypothèse nulle \(H_0\), la statistique de test \(W\) suit une distribution de Fisher-Snedecor à \(k - 1\) et \(N - k\) degrés de liberté.

\[W \sim F_{k - 1, N - k}\]

Preuves

Preuve du Théorème de Bartlett

La preuve du théorème de Bartlett repose sur l’approximation du logarithme des variances par une distribution normale. En utilisant le développement de Taylor, on peut montrer que la statistique \(B\) suit une distribution du chi-deux asymptotiquement. Les détails de cette preuve sont complexes et dépassent le cadre de cet article, mais l’idée principale est d’utiliser les propriétés des logarithmes et des variances pour établir la distribution asymptotique.

Preuve du Théorème de Levene

La preuve du théorème de Levene repose sur l’analyse des résidus absolutisés. En utilisant les propriétés des distributions \(F\), on peut montrer que la statistique \(W\) suit une distribution de Fisher-Snedecor sous l’hypothèse nulle. Cette preuve utilise également des approximations asymptotiques et des propriétés des résidus.

Propriétés et Corollaires

Propriétés du Test de Bartlett

  1. Le test de Bartlett est puissant lorsque les données suivent une distribution normale.

  2. Il est sensible aux violations de l’hypothèse de normalité.

  3. Il nécessite des échantillons de taille suffisante pour être fiable.

Propriétés du Test de Levene

  1. Le test de Levene est robuste face aux violations de l’hypothèse de normalité.

  2. Il est basé sur les résidus absolutisés, ce qui le rend moins sensible aux valeurs aberrantes.

  3. Il est particulièrement utile pour les échantillons de petite taille.

Corollaires

  1. Si le test de Bartlett rejette l’hypothèse nulle, il est recommandé d’utiliser des méthodes robustes pour l’analyse de variance.

  2. Si le test de Levene rejette l’hypothèse nulle, il est également recommandé d’utiliser des méthodes robustes pour l’analyse de variance.

  3. Les deux tests peuvent être utilisés en complément pour valider les résultats.

Conclusion

Le test d’égalité des variances est un outil essentiel dans l’analyse statistique. Les tests de Bartlett et de Levene offrent des approches complémentaires pour comparer les dispersions de plusieurs échantillons. Bien que le test de Bartlett soit puissant sous l’hypothèse de normalité, le test de Levene est plus robuste et adapté à des données non normales. Ces tests sont indispensables pour valider l’application d’autres procédures statistiques et assurer la fiabilité des résultats.
