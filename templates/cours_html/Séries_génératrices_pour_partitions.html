Séries génératrices pour partitions

Introduction et Motivations

L’étude des partitions d’entiers est un domaine riche de la combinatoire, intimement lié à de nombreuses branches des mathématiques. Les partitions d’un entier \(n\) sont les façons de l’écrire comme une somme de nombres entiers positifs, où l’ordre des termes n’est pas pris en compte. Par exemple, les partitions de 4 sont : \[1+1+1+1,\ 2+1+1,\ 2+2,\ 3+1,\ 4\] Les séries génératrices sont un outil puissant pour étudier les partitions. Elles permettent de codifier une grande quantité d’information sur les partitions dans une seule expression analytique, facilitant ainsi l’étude de leurs propriétés.

Les séries génératrices pour partitions émergent naturellement dans le cadre de la théorie des fonctions symétriques et des polynômes de Schur. Elles jouent un rôle central dans la théorie des représentations, en particulier pour les groupes symétriques et les algèbres de Lie. De plus, elles apparaissent dans divers problèmes de combinatoire énumérative et algébrique.

Définitions

Avant de définir les séries génératrices pour partitions, il est essentiel de comprendre ce que nous cherchons à capturer. Nous voulons une fonction qui encode le nombre de partitions d’un entier \(n\) en une seule expression. Cette fonction doit être capable de capturer non seulement le nombre total de partitions, mais aussi des propriétés plus fines, telles que la multiplicité des parties ou les contraintes sur les partitions.

Nous commençons par définir formellement une partition. Une partition d’un entier \(n\) est une suite décroissante d’entiers positifs \(\lambda_1 \geq \lambda_2 \geq \dots \geq \lambda_k > 0\) telle que \(\sum_{i=1}^k \lambda_i = n\). Le nombre de partitions de \(n\) est généralement noté \(p(n)\).

Nous définissons maintenant la série génératrice pour partitions. La série génératrice pour partitions est une fonction formelle en une variable \(q\) définie par : \[\sum_{n=0}^{\infty} p(n) q^n = \prod_{k=1}^{\infty} \frac{1}{1 - q^k}\] Cette série génératrice encode le nombre de partitions de chaque entier \(n\) dans les coefficients de la série.

Pour mieux comprendre cette définition, considérons ce que représente chaque facteur \(\frac{1}{1 - q^k}\) dans le produit infini. Le développement en série de \(\frac{1}{1 - q^k}\) est : \[\frac{1}{1 - q^k} = \sum_{m=0}^{\infty} q^{k m}\] Chaque terme \(q^{k m}\) représente une partie de taille \(k\) apparaissant \(m\) fois dans la partition. Le produit infini sur tous les \(k\) capture toutes les possibilités de partitions en sommant sur toutes les combinaisons possibles de parties.

Théorèmes

Un des théorèmes fondamentaux concernant les séries génératrices pour partitions est le théorème de Euler, qui donne une formule explicite pour la série génératrice des partitions.

La série génératrice pour partitions est donnée par : \[\sum_{n=0}^{\infty} p(n) q^n = \prod_{k=1}^{\infty} \frac{1}{1 - q^k}\]

Pour démontrer ce théorème, nous commençons par comprendre pourquoi le produit infini représente bien les partitions. Chaque facteur \(\frac{1}{1 - q^k}\) dans le produit correspond à la possibilité d’inclure des parties de taille \(k\) dans une partition. Le développement en série de \(\frac{1}{1 - q^k}\) montre que nous pouvons inclure zéro, une, deux, etc., parties de taille \(k\).

En multipliant tous ces facteurs ensemble, nous obtenons une série où chaque terme est de la forme \(q^n\), et le coefficient de \(q^n\) est précisément le nombre de partitions de \(n\). Cela provient du fait que chaque partition de \(n\) correspond à un choix unique de coefficients dans le produit des séries.

Preuves

Pour démontrer le théorème de Euler, nous procédons par induction sur \(n\). Nous montrons que pour tout entier positif \(n\), le coefficient de \(q^n\) dans \(\prod_{k=1}^{\infty} \frac{1}{1 - q^k}\) est égal au nombre de partitions de \(n\).

Nous commençons par la base de l’induction. Pour \(n=0\), il n’y a qu’une seule partition, qui est la partition vide. Le coefficient de \(q^0\) dans le produit infini est 1, ce qui correspond au nombre de partitions de 0.

Supposons maintenant que le théorème soit vrai pour tous les entiers inférieurs à \(n\). Nous devons montrer qu’il est vrai pour \(n\).

Considérons une partition quelconque de \(n\). Cette partition contient au moins une partie, disons de taille \(k\), où \(1 \leq k \leq n\). Nous pouvons alors écrire cette partition comme une somme de parties, dont l’une est \(k\) et les autres forment une partition de \(n - k\).

En utilisant le principe d’inclusion-exclusion, nous pouvons compter toutes les partitions de \(n\) en sommant sur toutes les possibilités de parties et en comptant les partitions des restes. Cela donne : \[p(n) = \sum_{k=1}^n p(n - k)\] où \(p(0) = 1\).

En utilisant la série génératrice, nous avons : \[\sum_{n=0}^{\infty} p(n) q^n = \sum_{n=0}^{\infty} \left( \sum_{k=1}^n p(n - k) \right) q^n = \left( \sum_{k=1}^{\infty} q^k \right) \left( \sum_{n=0}^{\infty} p(n) q^n \right)\] En utilisant le fait que \(\sum_{k=1}^{\infty} q^k = \frac{q}{1 - q}\), nous obtenons : \[\sum_{n=0}^{\infty} p(n) q^n = \frac{q}{1 - q} \sum_{n=0}^{\infty} p(n) q^n\] En multipliant les deux côtés par \((1 - q)\), nous obtenons : \[(1 - q) \sum_{n=0}^{\infty} p(n) q^n = q \sum_{n=0}^{\infty} p(n) q^n\] En réarrangeant, nous obtenons : \[\sum_{n=0}^{\infty} p(n) q^n = \frac{q}{1 - q} \sum_{n=0}^{\infty} p(n) q^n\] En utilisant le fait que \(\frac{1}{1 - q} = \sum_{m=0}^{\infty} q^m\), nous obtenons : \[\sum_{n=0}^{\infty} p(n) q^n = \left( \sum_{m=0}^{\infty} q^m \right) \left( \sum_{n=0}^{\infty} p(n) q^n \right)\] En développant le produit, nous obtenons : \[\sum_{n=0}^{\infty} p(n) q^n = \sum_{m=0}^{\infty} \sum_{n=0}^{\infty} p(n) q^{m + n}\] En changeant l’ordre des sommations, nous obtenons : \[\sum_{n=0}^{\infty} p(n) q^n = \sum_{k=0}^{\infty} \left( \sum_{m=0}^k p(k - m) \right) q^k\] En utilisant l’hypothèse d’induction, nous avons \(\sum_{m=0}^k p(k - m) = p(k)\). Donc : \[\sum_{n=0}^{\infty} p(n) q^n = \sum_{k=0}^{\infty} p(k) q^k\] Ce qui achève la preuve par induction.

Propriétés et Corollaires

Nous listons maintenant quelques propriétés importantes des séries génératrices pour partitions.

  1. La série génératrice pour partitions est une fonction holomorphe dans le disque unité \(|q| < 1\). Cela signifie que la série converge absolument pour ces valeurs de \(q\).

  2. La série génératrice pour partitions peut être exprimée en termes de fonctions thêta. En particulier, nous avons : \[\sum_{n=0}^{\infty} p(n) q^n = \prod_{k=1}^{\infty} (1 - q^{2k})(1 + q^{2k-1})^2\] Cette expression est utile pour étudier les propriétés analytiques de la série génératrice.

  3. La série génératrice pour partitions satisfait une relation de récurrence. En particulier, nous avons : \[p(n) = \sum_{k=1}^n (-1)^{k+1} \left( p(n - k) + p(n - 2k) \right)\] Cette relation est utile pour calculer les valeurs de \(p(n)\) de manière récursive.

Pour prouver la propriété (ii), nous utilisons l’identité suivante : \[\prod_{k=1}^{\infty} \frac{1}{1 - q^k} = \prod_{k=1}^{\infty} (1 - q^{2k})(1 + q^{2k-1})^2\] Cette identité peut être démontrée en utilisant les propriétés des fonctions thêta et des produits infinis.

Pour prouver la propriété (iii), nous utilisons le théorème de Euler et les relations de récurrence pour les coefficients des séries génératrices. En particulier, nous avons : \[p(n) = \sum_{k=1}^n (-1)^{k+1} \left( p(n - k) + p(n - 2k) \right)\] Cette relation peut être démontrée en utilisant le principe d’inclusion-exclusion et les propriétés des partitions.
