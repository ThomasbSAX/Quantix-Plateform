Taux de faux positifs (False Positive Rate)

Quantix

2026-01-11

Introduction et Motivations

Le taux de faux positifs, ou False Positive Rate (FPR), est une notion fondamentale en statistique et en apprentissage automatique. Il émerge dans le cadre de la classification binaire, où l’objectif est de distinguer deux classes distinctes. Le FPR mesure la proportion de cas négatifs qui sont incorrectement classés comme positifs.

Ce concept est indispensable dans de nombreux domaines, tels que la médecine, où un faux positif peut entraîner des diagnostics erronés et des traitements inutiles. Il est également crucial en sécurité informatique, où un faux positif peut déclencher des alertes inutiles et perturber les opérations.

Définitions

Pour définir le taux de faux positifs, commençons par comprendre ce que nous cherchons à mesurer. Supposons que nous avons un test de classification binaire qui prédit si une instance appartient à la classe positive ou négative. Nous voulons mesurer la proportion de cas négatifs qui sont incorrectement classés comme positifs.

Formellement, soit \(P\) l’ensemble des instances positives et \(N\) l’ensemble des instances négatives. Soit \(TP\) le nombre de vrais positifs, \(FP\) le nombre de faux positifs, \(TN\) le nombre de vrais négatifs et \(FN\) le nombre de faux négatifs.

Le taux de faux positifs est défini comme suit :

\[FPR = \frac{FP}{FP + TN}\]

Autrement dit, le FPR est la proportion de faux positifs parmi toutes les instances négatives.

Théorèmes

Un théorème important lié au taux de faux positifs est le théorème de Bayes, qui relie les probabilités conditionnelles. Ce théorème est souvent utilisé pour calculer le FPR dans des contextes probabilistes.

Supposons que nous avons un test de classification binaire avec une probabilité \(P(P)\) d’appartenir à la classe positive et \(P(N)\) d’appartenir à la classe négative. Supposons également que nous avons une probabilité \(P(T^+|P)\) de tester positif si l’instance est positive et \(P(T^-|N)\) de tester négatif si l’instance est négative.

Le théorème de Bayes nous permet de calculer la probabilité d’appartenir à la classe positive étant donné un test positif :

\[P(P|T^+) = \frac{P(T^+|P) P(P)}{P(T^+)}\]

où \(P(T^+) = P(T^+|P) P(P) + P(T^+|N) P(N)\).

Preuves

Pour prouver que le FPR est bien défini par la formule donnée, nous devons montrer que cette définition est cohérente avec les concepts de vrais positifs et faux négatifs.

Considérons un ensemble de données avec \(n\) instances, dont \(n_P\) sont positives et \(n_N\) sont négatives. Supposons que notre test de classification binaire classe correctement \(TP\) instances positives et \(TN\) instances négatives, tout en classant incorrectement \(FP\) instances négatives comme positives et \(FN\) instances positives comme négatives.

Nous avons donc :

\[TP + FN = n_P \quad \text{et} \quad FP + TN = n_N\]

Le taux de faux positifs est alors :

\[FPR = \frac{FP}{FP + TN} = \frac{FP}{n_N}\]

Cette formule montre que le FPR est bien la proportion de faux positifs parmi toutes les instances négatives.

Propriétés et Corollaires

Le taux de faux positifs présente plusieurs propriétés intéressantes :

  • Le FPR est toujours compris entre 0 et 1, car il s’agit d’une proportion.

  • Un FPR élevé indique que le test de classification a tendance à classer incorrectement les instances négatives comme positives.

  • Le FPR est inversement lié au taux de vrais négatifs (True Negative Rate, TNR), qui est défini comme \(TNR = \frac{TN}{FP + TN}\).

Pour prouver la propriété (iii), nous avons :

\[FPR + TNR = \frac{FP}{FP + TN} + \frac{TN}{FP + TN} = 1\]

Cette relation montre que le FPR et le TNR sont complémentaires.
