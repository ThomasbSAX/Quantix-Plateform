Taux de vrais positifs (True Positive Rate)

Quantix

2026-01-11

Introduction et Motivations

Le Taux de vrais positifs (True Positive Rate, ou TPR) est une mesure fondamentale en statistique et en apprentissage automatique, particulièrement dans le cadre de la classification binaire. Il émerge comme une réponse à la nécessité d’évaluer la performance des modèles prédictifs, en fournissant une métrique qui capture leur capacité à identifier correctement les instances positives.

Historiquement, le TPR trouve ses racines dans la théorie des tests statistiques et des fonctions de décision. Il est indispensable dans les domaines où les faux négatifs sont coûteux, comme le diagnostic médical ou la détection de fraudes. Le TPR permet de quantifier l’efficacité d’un modèle à minimiser ces erreurs critiques.

Définitions

Pour introduire le Taux de vrais positifs, considérons un modèle de classification binaire qui prédit si une instance appartient à la classe positive ou négative. Nous cherchons à mesurer la proportion d’instances positives correctement identifiées par le modèle.

Formellement, soit \(\mathcal{D}\) un ensemble de données étiquetées, où chaque instance \(x_i \in \mathcal{D}\) est associée à une étiquette \(y_i \in \{0, 1\}\), avec \(y_i = 1\) pour la classe positive et \(y_i = 0\) pour la classe négative. Soit \(\hat{y}_i\) la prédiction du modèle pour l’instance \(x_i\).

Le Taux de vrais positifs est défini comme la proportion d’instances positives correctement prédites :

Le Taux de vrais positifs (TPR) est donné par : \[\text{TPR} = \frac{\text{VP}}{\text{VP} + \text{FN}}\] où VP (Vrais Positifs) est le nombre d’instances positives correctement prédites, et FN (Faux Négatifs) est le nombre d’instances positives incorrectement prédites.

En termes quantifiés : \[\text{TPR} = \frac{\sum_{i=1}^{N} \mathbb{I}(\hat{y}_i = 1 \text{ et } y_i = 1)}{\sum_{i=1}^{N} \mathbb{I}(y_i = 1)}\] où \(\mathbb{I}\) est la fonction indicatrice et \(N\) est le nombre total d’instances.

Théorèmes et Propriétés

Le TPR est lié à d’autres métriques de performance, telles que la sensibilité et le rappel. En effet, le TPR est souvent utilisé de manière interchangeable avec ces termes.

Le Taux de vrais positifs est équivalent à la sensibilité d’un modèle de classification binaire. La sensibilité mesure la capacité du modèle à identifier les instances positives.

Formellement, si \(\text{Sensibilité}\) est définie comme : \[\text{Sensibilité} = \frac{\text{VP}}{\text{VP} + \text{FN}}\] alors : \[\text{TPR} = \text{Sensibilité}\]

Proof. La preuve découle directement des définitions. La sensibilité est calculée comme la proportion d’instances positives correctement prédites par rapport au nombre total d’instances positives. Le TPR est défini de manière identique, ce qui établit l’égalité. ◻

Propriétés et Corollaires

Le TPR possède plusieurs propriétés importantes qui en font une métrique clé dans l’évaluation des modèles de classification.

Le TPR est une mesure comprise entre 0 et 1, où :

  • Un TPR de 1 indique que toutes les instances positives sont correctement identifiées.

  • Un TPR de 0 indique que aucune instance positive n’est correctement identifiée.

Proof. Par définition, le TPR est le rapport entre deux quantités positives ou nulles. Le numérateur (VP) et le dénominateur (VP + FN) sont tous deux non négatifs. De plus, si VP = 0, alors TPR = 0, et si FN = 0, alors TPR = 1. ◻

Le TPR est maximisé lorsque le nombre de faux négatifs (FN) est minimisé. Cela signifie que pour améliorer le TPR, il est crucial de réduire les erreurs de classification des instances positives.

Conclusion

Le Taux de vrais positifs est une métrique essentielle pour évaluer la performance des modèles de classification binaire. Il capture la capacité d’un modèle à identifier correctement les instances positives, ce qui est crucial dans de nombreuses applications pratiques. En comprenant et en optimisant le TPR, les chercheurs et les praticiens peuvent améliorer la fiabilité et l’efficacité de leurs modèles prédictifs.
