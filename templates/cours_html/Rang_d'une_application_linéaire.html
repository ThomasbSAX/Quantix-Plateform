Rang d’une application linéaire

Quantix

2026-01-11

Introduction et Motivations

L’étude des applications linéaires est au cœur de l’algèbre linéaire. Parmi les nombreux concepts associés, le rang d’une application linéaire occupe une place centrale. Introduit au début du XXème siècle, ce concept permet de mesurer la dimension de l’image d’une application linéaire. Cette notion est indispensable pour comprendre les propriétés fondamentales des applications linéaires, telles que l’injectivité et la surjectivité. Le rang permet également de généraliser des résultats classiques comme le théorème du rang, qui relie les dimensions de l’image et du noyau d’une application linéaire.

Définitions

Pour introduire la notion de rang, commençons par rappeler quelques concepts préliminaires. Considérons un espace vectoriel \(E\) sur un corps commutatif \(K\). Une application linéaire \(f: E \rightarrow F\) est une fonction qui préserve les opérations de l’addition et de la multiplication par un scalaire.

Soit \(f: E \rightarrow F\) une application linéaire. L’image de \(f\), notée \(\text{Im}(f)\), est l’ensemble des vecteurs de \(F\) qui sont l’image d’au moins un vecteur de \(E\). Formellement, \[\text{Im}(f) = \{ f(x) \in F \mid x \in E \}.\]

Le rang d’une application linéaire est alors défini comme la dimension de son image.

Soit \(f: E \rightarrow F\) une application linéaire. Le rang de \(f\), noté \(\text{rang}(f)\), est la dimension de l’image de \(f\). Autrement dit, \[\text{rang}(f) = \dim(\text{Im}(f)).\]

Théorèmes

Le théorème du rang est l’un des résultats les plus importants en algèbre linéaire. Il établit une relation fondamentale entre le rang d’une application linéaire et la dimension de son noyau.

Soit \(f: E \rightarrow F\) une application linéaire. Alors, \[\dim(E) = \text{rang}(f) + \dim(\ker(f)),\] où \(\ker(f)\) désigne le noyau de \(f\), c’est-à-dire l’ensemble des vecteurs de \(E\) qui sont envoyés sur le vecteur nul de \(F\).

Preuves

Pour démontrer le théorème du rang, nous allons utiliser plusieurs propriétés fondamentales des espaces vectoriels. Commençons par rappeler que le noyau de \(f\) est un sous-espace vectoriel de \(E\).

Preuve du théorème du rang. Soit \(f: E \rightarrow F\) une application linéaire. Nous allons montrer que \[\dim(E) = \text{rang}(f) + \dim(\ker(f)).\]

1. **Construction d’une base adaptée**: Considérons une base \((e_1, \ldots, e_k)\) de \(\ker(f)\). Nous pouvons compléter cette base pour obtenir une base de \(E\). Supposons que \((e_1, \ldots, e_k, e_{k+1}, \ldots, e_n)\) soit une base de \(E\).

2. **Image des vecteurs de la base**: Montrons que les images \(f(e_{k+1}), \ldots, f(e_n)\) forment une base de \(\text{Im}(f)\).

- **Linéaire indépendance**: Supposons que \(\sum_{i=k+1}^n \lambda_i f(e_i) = 0\). Alors, \(f\left(\sum_{i=k+1}^n \lambda_i e_i\right) = 0\). Puisque \((e_1, \ldots, e_n)\) est une base de \(E\), il existe des scalaires \(\mu_1, \ldots, \mu_k\) tels que \(\sum_{i=k+1}^n \lambda_i e_i = -\sum_{i=1}^k \mu_i e_i\). Cependant, comme \((e_1, \ldots, e_k)\) est une base de \(\ker(f)\), nous avons \(f\left(\sum_{i=1}^k \mu_i e_i\right) = 0\). Par conséquent, \(f\left(\sum_{i=k+1}^n \lambda_i e_i\right) = 0\). Mais \(f\) est linéaire, donc \(\sum_{i=k+1}^n \lambda_i f(e_i) = 0\). Ainsi, \(\lambda_{k+1} = \ldots = \lambda_n = 0\), ce qui montre que \(f(e_{k+1}), \ldots, f(e_n)\) sont linéairement indépendants.

- **Génération de l’image**: Tout vecteur \(y \in \text{Im}(f)\) peut s’écrire comme \(y = f(x)\) pour un certain \(x \in E\). Puisque \((e_1, \ldots, e_n)\) est une base de \(E\), il existe des scalaires \(\alpha_1, \ldots, \alpha_n\) tels que \(x = \sum_{i=1}^n \alpha_i e_i\). Alors, \(y = f(x) = \sum_{i=1}^n \alpha_i f(e_i) = \sum_{i=k+1}^n \alpha_i f(e_i)\), car \(f(e_1) = \ldots = f(e_k) = 0\). Ainsi, \(y\) est une combinaison linéaire de \(f(e_{k+1}), \ldots, f(e_n)\).

3. **Conclusion**: Puisque \(f(e_{k+1}), \ldots, f(e_n)\) forment une base de \(\text{Im}(f)\), nous avons \(\text{rang}(f) = n - k\). Mais \(\dim(\ker(f)) = k\), donc \[\dim(E) = n = k + (n - k) = \dim(\ker(f)) + \text{rang}(f).\] ◻

Propriétés et Corollaires

Le théorème du rang a de nombreuses conséquences importantes. Nous en énumérons quelques-unes ci-dessous.

  1. **Injectivité et rang**: Une application linéaire \(f: E \rightarrow F\) est injective si et seulement si \(\text{rang}(f) = \dim(E)\).

    Proof. Supposons que \(f\) soit injective. Alors, \(\ker(f) = \{0\}\), donc \(\dim(\ker(f)) = 0\). Par le théorème du rang, nous avons \(\dim(E) = \text{rang}(f) + 0\), donc \(\text{rang}(f) = \dim(E)\).

    Réciproquement, supposons que \(\text{rang}(f) = \dim(E)\). Par le théorème du rang, nous avons \(\dim(\ker(f)) = 0\), donc \(\ker(f) = \{0\}\). Ainsi, \(f\) est injective. ◻

  2. **Surjectivité et rang**: Une application linéaire \(f: E \rightarrow F\) est surjective si et seulement si \(\text{rang}(f) = \dim(F)\).

    Proof. Supposons que \(f\) soit surjective. Alors, \(\text{Im}(f) = F\), donc \(\text{rang}(f) = \dim(F)\).

    Réciproquement, supposons que \(\text{rang}(f) = \dim(F)\). Alors, \(\text{Im}(f)\) est un sous-espace vectoriel de \(F\) de dimension \(\dim(F)\), donc \(\text{Im}(f) = F\). Ainsi, \(f\) est surjective. ◻

  3. **Rang et composition**: Soient \(f: E \rightarrow F\) et \(g: F \rightarrow G\) des applications linéaires. Alors, \[\text{rang}(g \circ f) \leq \min(\text{rang}(f), \text{rang}(g)).\]

    Proof. Nous savons que \(\text{rang}(g \circ f) = \dim(\text{Im}(g \circ f))\). Mais \(\text{Im}(g \circ f) \subseteq \text{Im}(g)\), donc \(\dim(\text{Im}(g \circ f)) \leq \dim(\text{Im}(g)) = \text{rang}(g)\). De même, \(\text{Im}(g \circ f) \subseteq \text{Im}(f)\), donc \(\dim(\text{Im}(g \circ f)) \leq \dim(\text{Im}(f)) = \text{rang}(f)\). Ainsi, \(\text{rang}(g \circ f) \leq \min(\text{rang}(f), \text{rang}(g))\). ◻

Conclusion

Le rang d’une application linéaire est un concept fondamental en algèbre linéaire. Il permet de mesurer la dimension de l’image d’une application linéaire et est étroitement lié à la dimension du noyau. Le théorème du rang établit une relation fondamentale entre ces deux dimensions, ce qui permet de déduire de nombreuses propriétés importantes des applications linéaires. Les corollaires du théorème du rang fournissent des critères simples pour déterminer si une application linéaire est injective ou surjective. Enfin, la propriété de composition montre comment le rang se comporte sous composition d’applications linéaires.
