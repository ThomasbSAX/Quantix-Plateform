Distance de Gower pour variables discrètes symétriques

Quantix

2026-01-11

Introduction et Motivations

La distance de Gower émerge dans le cadre de l’analyse des données multivariées, où l’on cherche à mesurer la dissimilarité entre objets décrits par des variables de natures diverses. Son origine historique remonte aux travaux de Gower (1971), qui a proposé une généralisation des distances classiques pour prendre en compte des variables qualitatives, ordinales et quantitatives. Cette notion est indispensable dans les contextes où les données sont hétérogènes, par exemple en biologie pour comparer des espèces ou en sociologie pour analyser des enquêtes.

Définitions

Pour introduire la distance de Gower, commençons par comprendre ce que nous cherchons à mesurer. Imaginons deux objets décrits par plusieurs variables discrètes symétriques, c’est-à-dire que les modalités de chaque variable sont équivalentes et n’ont pas d’ordre naturel. Nous voulons quantifier la dissimilarité entre ces deux objets en tenant compte de toutes les variables.

Formellement, soit \(X\) un ensemble d’objets et \(V = \{v_1, v_2, \ldots, v_p\}\) un ensemble de variables discrètes symétriques. Pour deux objets \(x_i, x_j \in X\), la distance de Gower est définie comme suit:

\[d_{ij} = \frac{1}{p} \sum_{k=1}^{p} d_{ij}^{(k)}\]

où \(d_{ij}^{(k)}\) est la dissimilarité entre les objets \(x_i\) et \(x_j\) pour la variable \(v_k\), définie par:

\[d_{ij}^{(k)} = \begin{cases} 0 & \text{si } x_i^{(k)} = x_j^{(k)}, \\ 1 & \text{sinon.} \end{cases}\]

En d’autres termes, pour chaque variable, nous attribuons une dissimilarité de 0 si les modalités sont identiques et 1 sinon. La distance globale est la moyenne de ces dissimilarités sur toutes les variables.

Théorèmes

Considérons maintenant quelques propriétés importantes de la distance de Gower. Nous voulons montrer que cette distance satisfait les axiomes d’une distance, c’est-à-dire qu’elle est définie positive, symétrique et vérifie l’inégalité triangulaire.

Soit \(d\) la distance de Gower définie sur un ensemble \(X\) avec des variables discrètes symétriques. Alors:

  1. \(d_{ij} \geq 0\) pour tout \(i, j\).

  2. \(d_{ij} = 0\) si et seulement si \(i = j\).

  3. \(d_{ij} = d_{ji}\) pour tout \(i, j\).

  4. \(d_{ik} \leq d_{ij} + d_{jk}\) pour tout \(i, j, k\).

Preuves

Proof.

  • Puisque \(d_{ij}^{(k)} \in \{0, 1\}\), il est clair que \(d_{ij} = \frac{1}{p} \sum_{k=1}^{p} d_{ij}^{(k)} \geq 0\).

  • Si \(i = j\), alors \(d_{ij}^{(k)} = 0\) pour tout \(k\), donc \(d_{ij} = 0\). Réciproquement, si \(d_{ij} = 0\), alors \(d_{ij}^{(k)} = 0\) pour tout \(k\), ce qui implique que \(x_i^{(k)} = x_j^{(k)}\) pour tout \(k\), donc \(i = j\).

  • Par définition, \(d_{ij}^{(k)} = d_{ji}^{(k)}\) pour tout \(k\), donc \(d_{ij} = d_{ji}\).

  • L’inégalité triangulaire pour la distance de Gower est une conséquence directe des propriétés des variables discrètes symétriques. En effet, pour chaque variable \(v_k\), nous avons: \[d_{ik}^{(k)} + d_{kj}^{(k)} \geq d_{ij}^{(k)}\] En sommant sur toutes les variables et en divisant par \(p\), nous obtenons: \[d_{ik} + d_{kj} \geq d_{ij}\]

 ◻

Propriétés et Corollaires

  1. La distance de Gower est invariante par permutation des variables. En effet, si nous permutons les variables, la somme \(\sum_{k=1}^{p} d_{ij}^{(k)}\) reste inchangée, donc \(d_{ij}\) est également inchangé.

  2. La distance de Gower peut être généralisée à des variables pondérées. Si chaque variable \(v_k\) a un poids \(w_k\), alors la distance pondérée est définie par: \[d_{ij} = \frac{\sum_{k=1}^{p} w_k d_{ij}^{(k)}}{\sum_{k=1}^{p} w_k}\]

  3. La distance de Gower est particulièrement utile pour les variables discrètes symétriques, mais elle peut également être adaptée à d’autres types de variables en définissant des dissimilarités appropriées pour chaque type.

Conclusion

La distance de Gower est un outil puissant pour mesurer la dissimilarité entre objets décrits par des variables discrètes symétriques. Ses propriétés mathématiques en font un choix naturel pour l’analyse des données multivariées hétérogènes.
