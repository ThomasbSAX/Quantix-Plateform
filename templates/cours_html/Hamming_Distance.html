Hamming Distance: A Fundamental Concept in Coding Theory

Introduction and Motivations

The Hamming distance, named after the mathematician Richard Hamming, is a pivotal concept in coding theory and information theory. Originating from the need to detect and correct errors in digital data transmission, the Hamming distance provides a metric to quantify the difference between two strings of equal length. This concept is indispensable in various applications, including error-correcting codes, cryptography, and bioinformatics.

The emergence of the Hamming distance can be traced back to the early days of computer science and telecommunications. As data transmission became more prevalent, the need to ensure data integrity over noisy channels became paramount. Definitions

To understand the Hamming distance, let us first consider two strings of equal length. We aim to find a measure that quantifies how different these strings are. Intuitively, we can count the number of positions at which the corresponding symbols in the strings differ.

Formally, let \(\mathbb{F}_q^n\) denote the set of all strings of length \(n\) over a finite field \(\mathbb{F}_q\). The Hamming distance between two strings \(\mathbf{x} = (x_1, x_2, \ldots, x_n)\) and \(\mathbf{y} = (y_1, y_2, \ldots, y_n)\) in \(\mathbb{F}_q^n\) is defined as:

\[d_H(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^n \delta(x_i, y_i)\]

where \(\delta(a, b) = 1\) if \(a \neq b\) and \(\delta(a, b) = 0\) if \(a = b\).

Alternatively, the Hamming distance can be expressed using indicator functions:

\[d_H(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^n \mathbb{I}(x_i \neq y_i)\]

where \(\mathbb{I}\) is the indicator function.

Theorems

One of the fundamental theorems related to the Hamming distance is the Singleton Bound, which provides an upper limit on the size of a code given its minimum Hamming distance.

Let \(C\) be a code of length \(n\) and size \(M\), with minimum Hamming distance \(d\). The Singleton Bound states that:

\[M \leq q^{n - d + 1}\]

This theorem is a consequence of the fact that any two distinct codewords must differ in at least \(d\) positions.

Another important theorem is the Hamming Bound, which provides a condition for a code to be perfect. A code \(C\) is perfect if the Hamming spheres of radius \(t\) around each codeword cover the entire space without overlapping.

The Hamming Bound states that for a \(t\)-error-correcting code of length \(n\) and size \(M\), the following inequality must hold:

\[M \leq \frac{q^n}{\sum_{i=0}^t \binom{n}{i} (q - 1)^i}\]

Proofs

Let us prove the Singleton Bound. Consider two distinct codewords \(\mathbf{x}\) and \(\mathbf{y}\) in \(C\). By definition, the Hamming distance \(d_H(\mathbf{x}, \mathbf{y}) \geq d\).

Now, consider the linear transformation \(T: \mathbb{F}_q^n \rightarrow \mathbb{F}_q^{n - d + 1}\) defined by:

\[T(\mathbf{z}) = (z_1, z_2, \ldots, z_{n - d + 1})\]

This transformation maps any string \(\mathbf{z}\) to its first \(n - d + 1\) symbols. Since \(d_H(\mathbf{x}, \mathbf{y}) \geq d\), the first \(n - d + 1\) symbols of \(\mathbf{x}\) and \(\mathbf{y}\) must differ in at least one position. Therefore, the transformation \(T\) is injective when restricted to \(C\).

By the pigeonhole principle, the size of \(C\) must satisfy:

\[M \leq q^{n - d + 1}\]

This completes the proof of the Singleton Bound.

Properties and Corollaries

The Hamming distance exhibits several important properties:

  1. The Hamming distance is a metric. It satisfies the following properties for any strings \(\mathbf{x}, \mathbf{y}, \mathbf{z} \in \mathbb{F}_q^n\): \[d_H(\mathbf{x}, \mathbf{y}) \geq 0\] \[d_H(\mathbf{x}, \mathbf{y}) = 0 \iff \mathbf{x} = \mathbf{y}\] \[d_H(\mathbf{x}, \mathbf{y}) = d_H(\mathbf{y}, \mathbf{x})\] \[d_H(\mathbf{x}, \mathbf{z}) \leq d_H(\mathbf{x}, \mathbf{y}) + d_H(\mathbf{y}, \mathbf{z})\]

  2. The Hamming distance is invariant under coordinate permutations. That is, for any permutation \(\sigma\) of \(\{1, 2, \ldots, n\}\), we have: \[d_H(\mathbf{x}, \mathbf{y}) = d_H((x_{\sigma(1)}, x_{\sigma(2)}, \ldots, x_{\sigma(n)}), (y_{\sigma(1)}, y_{\sigma(2)}, \ldots, y_{\sigma(n)}))\]

  3. The Hamming distance can be used to define the concept of a sphere in \(\mathbb{F}_q^n\). The Hamming sphere of radius \(t\) around a string \(\mathbf{x}\) is the set of all strings within Hamming distance \(t\) from \(\mathbf{x}\): \[S(\mathbf{x}, t) = \{ \mathbf{y} \in \mathbb{F}_q^n \mid d_H(\mathbf{x}, \mathbf{y}) \leq t \}\]

Conclusion

The Hamming distance is a fundamental concept in coding theory, with applications ranging from error-correcting codes to bioinformatics. Its properties and related theorems provide powerful tools for designing efficient and reliable data transmission systems. As research in coding theory continues to advance, the Hamming distance remains a cornerstone in this ever-evolving field.
