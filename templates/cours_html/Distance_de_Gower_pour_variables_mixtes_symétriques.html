Distance de Gower pour variables mixtes symétriques

Quantix

2026-01-11

Introduction et Motivations

La distance de Gower émerge dans le contexte de l’analyse des données multivariées, où les ensembles de variables sont souvent mixtes, combinant à la fois des variables quantitatives et qualitatives. L’origine de cette distance remonte aux travaux de Gower (1971), qui cherchait à généraliser les notions de similarité et de dissimilarité pour des jeux de données hétérogènes. Cette notion est indispensable dans les domaines où l’analyse statistique nécessite une mesure robuste et flexible, capable de capturer la complexité des relations entre variables de nature différente.

L’intérêt principal de la distance de Gower réside dans sa capacité à traiter des variables symétriques, c’est-à-dire des variables qualitatives sans ordre intrinsèque. En intégrant ces variables dans un cadre métrique cohérent, la distance de Gower permet d’effectuer des analyses telles que les classifications hiérarchiques ou les régressions multivariées, même en présence de données manquantes.

Définitions

Pour définir la distance de Gower, nous cherchons à mesurer la dissimilarité entre deux objets décrits par un ensemble de variables mixtes. L’idée est d’unifier les contributions des différentes variables, qu’elles soient quantitatives ou qualitatives, en une seule mesure globale.

Supposons que nous avons un ensemble de \(n\) objets décrits par \(p\) variables. Pour chaque paire d’objets \(i\) et \(j\), nous voulons calculer une distance \(d_{ij}\).

Soit \(X\) une matrice \(n \times p\) contenant les données, où chaque colonne représente une variable et chaque ligne un objet. Pour deux objets \(i\) et \(j\), la distance de Gower est définie comme :

\[d_{ij} = \frac{\sum_{k=1}^{p} \delta_{ijk}}{p}\]

où \(\delta_{ijk}\) est la contribution de la variable \(k\) à la distance entre les objets \(i\) et \(j\).

Pour les variables quantitatives, la contribution est simplement la différence normalisée :

\[\delta_{ijk} = \frac{|x_{ik} - x_{jk}|}{\max_k x_k - \min_k x_k}\]

Pour les variables qualitatives symétriques, la contribution est définie comme :

\[\delta_{ijk} = \begin{cases} 0 & \text{si } x_{ik} = x_{jk} \\ 1 & \text{sinon} \end{cases}\]

Théorèmes

Un théorème fondamental lié à la distance de Gower est celui de sa métrique. Nous cherchons à montrer que cette distance satisfait les propriétés d’une distance métrique.

La distance de Gower \(d_{ij}\) satisfait les propriétés suivantes pour tout triplet d’objets \(i, j, k\) :

  1. Positivité : \(d_{ij} \geq 0\) et \(d_{ij} = 0\) si et seulement si \(i = j\).

  2. Symétrie : \(d_{ij} = d_{ji}\).

  3. Inégalité triangulaire : \(d_{ij} \leq d_{ik} + d_{kj}\).

Preuves

Pour prouver la positivité, nous observons que \(\delta_{ijk} \geq 0\) par définition, et \(d_{ij} = 0\) seulement si tous les \(\delta_{ijk} = 0\), ce qui implique que \(i = j\).

La symétrie découle directement de la symétrie des contributions \(\delta_{ijk}\).

Pour l’inégalité triangulaire, nous utilisons le fait que les contributions \(\delta_{ijk}\) sont bornées par 1. Ainsi, pour toute variable \(k\), nous avons :

\[\delta_{ijk} \leq \delta_{ikm} + \delta_{mjk}\]

En sommant sur toutes les variables et en divisant par \(p\), nous obtenons l’inégalité triangulaire pour \(d_{ij}\).

Propriétés et Corollaires

  1. Invariance par transformation linéaire : La distance de Gower est invariante par toute transformation linéaire des variables quantitatives.

  2. Gestion des données manquantes : La distance de Gower peut être adaptée pour gérer les données manquantes en excluant simplement les variables manquantes du calcul de la distance.

  3. Extension aux variables ordinales : Pour les variables qualitatives ordinales, la contribution peut être définie comme :

    \[\delta_{ijk} = \frac{|x_{ik} - x_{jk}|}{\max_k x_k - \min_k x_k}\]

    où \(x_{ik}\) et \(x_{jk}\) sont les rangs des modalités.

Conclusion

La distance de Gower offre une solution élégante et puissante pour l’analyse des données mixtes. Son utilisation permet de surmonter les limitations des méthodes traditionnelles et d’ouvrir de nouvelles perspectives dans l’exploration des jeux de données complexes.
