Identités de Chebyshev trigonométriques

Introduction et Motivations

Les identités de Chebyshev trigonométriques constituent un ensemble fondamental d’équations reliant les fonctions trigonométriques entre elles. Ces identités trouvent leur origine dans les travaux de Pafnuty Chebyshev, un mathématicien russe du XIXème siècle. Elles jouent un rôle crucial dans l’analyse harmonique, la théorie des équations différentielles et de nombreuses applications en physique et en ingénierie.

Ces identités émergent naturellement lorsque l’on cherche à simplifier des expressions trigonométriques complexes ou à résoudre des équations différentielles non linéaires. Elles sont indispensables pour transformer des produits de fonctions trigonométriques en sommes, ce qui facilite grandement les calculs et les démonstrations.

Définitions

Avant d’énoncer les identités de Chebyshev, il est essentiel de rappeler quelques définitions fondamentales concernant les fonctions trigonométriques.

Définition des fonctions trigonométriques

Les fonctions trigonométriques sont définies comme suit pour tout réel \(x\) :

Soit \(x \in \mathbb{R}\). On définit : \[\sin(x) = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n+1}}{(2n+1)!}\] \[\cos(x) = \sum_{n=0}^{\infty} \frac{(-1)^n x^{2n}}{(2n)!}\]

Propriétés de base

Les fonctions trigonométriques satisfont les propriétés suivantes :

Pour tout \(x, y \in \mathbb{R}\), on a : \[\sin(x + y) = \sin x \cos y + \cos x \sin y\] \[\cos(x + y) = \cos x \cos y - \sin x \sin y\]

Identités de Chebyshev

Les identités de Chebyshev permettent de transformer des produits de fonctions trigonométriques en sommes. Elles sont particulièrement utiles pour simplifier les expressions et résoudre des équations différentielles.

Identité de Chebyshev pour le produit de sinus

Pour tout \(x, y \in \mathbb{R}\), on a : \[2 \sin x \sin y = \cos(x - y) - \cos(x + y)\]

Proof. Partons de l’expression \(\cos(x - y) - \cos(x + y)\). En utilisant les formules de trigonométrie, on obtient : \[\cos(x - y) = \cos x \cos y + \sin x \sin y\] \[\cos(x + y) = \cos x \cos y - \sin x \sin y\] En soustrayant ces deux équations, on obtient : \[\cos(x - y) - \cos(x + y) = 2 \sin x \sin y\] Ce qui démontre l’identité. ◻

Identité de Chebyshev pour le produit de cosinus

Pour tout \(x, y \in \mathbb{R}\), on a : \[2 \cos x \cos y = \cos(x - y) + \cos(x + y)\]

Proof. Partons de l’expression \(\cos(x - y) + \cos(x + y)\). En utilisant les formules de trigonométrie, on obtient : \[\cos(x - y) = \cos x \cos y + \sin x \sin y\] \[\cos(x + y) = \cos x \cos y - \sin x \sin y\] En additionnant ces deux équations, on obtient : \[\cos(x - y) + \cos(x + y) = 2 \cos x \cos y\] Ce qui démontre l’identité. ◻

Identité de Chebyshev pour le produit sinus-cosinus

Pour tout \(x, y \in \mathbb{R}\), on a : \[2 \sin x \cos y = \sin(x + y) + \sin(x - y)\]

Proof. Partons de l’expression \(\sin(x + y) + \sin(x - y)\). En utilisant les formules de trigonométrie, on obtient : \[\sin(x + y) = \sin x \cos y + \cos x \sin y\] \[\sin(x - y) = \sin x \cos y - \cos x \sin y\] En additionnant ces deux équations, on obtient : \[\sin(x + y) + \sin(x - y) = 2 \sin x \cos y\] Ce qui démontre l’identité. ◻

Propriétés et Corollaires

Les identités de Chebyshev permettent de démontrer plusieurs propriétés intéressantes concernant les fonctions trigonométriques.

Propriété (i) : Symétrie

Pour tout \(x, y \in \mathbb{R}\), on a : \[2 \sin x \sin y = 2 \sin y \sin x\]

Proof. Cette propriété découle directement de la commutativité du produit : \[2 \sin x \sin y = 2 \sin y \sin x\] ◻

Propriété (ii) : Linéarité

Pour tout \(x, y, z \in \mathbb{R}\), on a : \[2 \sin(x + y) \sin z = \cos(x - z) - \cos(x + z) + \cos(y - z) - \cos(y + z)\]

Proof. En utilisant l’identité de Chebyshev pour le produit de sinus et la linéarité des fonctions trigonométriques, on obtient : \[2 \sin(x + y) \sin z = 2 (\sin x \cos y + \cos x \sin y) \sin z\] \[= 2 \sin x \cos y \sin z + 2 \cos x \sin y \sin z\] \[= \cos(x - z) - \cos(x + z) + \cos(y - z) - \cos(y + z)\] ◻

Conclusion

Les identités de Chebyshev trigonométriques sont des outils puissants pour simplifier les expressions trigonométriques et résoudre des équations différentielles. Elles trouvent leurs applications dans de nombreux domaines, allant de l’analyse harmonique à la physique théorique. Leur étude approfondie permet de mieux comprendre les propriétés fondamentales des fonctions trigonométriques et d’en exploiter pleinement le potentiel.
