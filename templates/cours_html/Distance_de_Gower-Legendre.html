Distance de Gower-Legendre : Une Mesure Multidimensionnelle pour les Données Mixtes

Quantix

2026-01-11

Introduction et Motivations

L’analyse des données multidimensionnelles est un défi fondamental en statistique et en apprentissage automatique. Les données réelles sont souvent composées de variables de types différents : quantitatives, ordinales et nominales. La distance de Gower-Legendre émerge comme une solution élégante pour mesurer la dissimilarité entre des objets décrits par de telles variables mixtes.

Historiquement, cette distance s’inscrit dans la lignée des travaux de Gower (1971) sur les mesures de dissimilarité pour données mixtes, et Legendre et Legendre (1983) sur l’analyse des tableaux écologiques. Elle est indispensable dans les contextes où l’hétérogénéité des variables rend inapplicables les distances classiques comme la distance euclidienne.

Définitions

Pour définir la distance de Gower-Legendre, considérons un ensemble d’objets décrits par des variables de types différents. Nous cherchons une mesure qui capture la dissimilarité entre ces objets, en tenant compte de la nature des variables.

Supposons que nous avons un ensemble \(\mathcal{O}\) d’objets et un ensemble \(\mathcal{V} = \mathcal{V}_Q \cup \mathcal{V}_O \cup \mathcal{V}_N\) de variables, où \(\mathcal{V}_Q\), \(\mathcal{V}_O\) et \(\mathcal{V}_N\) représentent respectivement les variables quantitatives, ordinales et nominales.

Nous voulons une fonction \(d: \mathcal{O} \times \mathcal{O} \rightarrow \mathbb{R}^+\) qui mesure la dissimilarité entre deux objets \(o_i, o_j \in \mathcal{O}\). Cette fonction doit être symétrique (\(d(o_i, o_j) = d(o_j, o_i)\)) et vérifier \(d(o_i, o_i) = 0\).

Soient \(o_i, o_j \in \mathcal{O}\) deux objets décrits par un ensemble de variables \(\mathcal{V}\). La distance de Gower-Legendre entre \(o_i\) et \(o_j\) est définie comme :

\[d_{GL}(o_i, o_j) = \frac{\sum_{k=1}^{p} w_k d_k(o_i, o_j)}{\sum_{k=1}^{p} w_k}\]

où \(p\) est le nombre total de variables, \(w_k\) est le poids associé à la variable \(k\), et \(d_k(o_i, o_j)\) est la dissimilarité entre les valeurs des objets \(o_i\) et \(o_j\) pour la variable \(k\).

Pour les variables quantitatives, la dissimilarité est définie par :

\[d_k(o_i, o_j) = \frac{|o_{ik} - o_{jk}|}{R_k}\]

où \(R_k\) est l’étendue de la variable \(k\).

Pour les variables ordinales, la dissimilarité est définie par :

\[d_k(o_i, o_j) = \frac{|r_{ik} - r_{jk}|}{M_k - 1}\]

où \(r_{ik}\) et \(r_{jk}\) sont les rangs des valeurs des objets \(o_i\) et \(o_j\) pour la variable \(k\), et \(M_k\) est le nombre de modalités de la variable \(k\).

Pour les variables nominales, la dissimilarité est définie par :

\[d_k(o_i, o_j) = \begin{cases} 0 & \text{si } o_{ik} = o_{jk}, \\ 1 & \text{sinon.} \end{cases}\]

Théorèmes

Nous présentons maintenant un théorème fondamental concernant la distance de Gower-Legendre.

La distance de Gower-Legendre \(d_{GL}\) est une distance au sens mathématique si et seulement si les poids \(w_k\) sont positifs et les dissimilarités \(d_k\) vérifient les propriétés suivantes :

  1. Symétrie : \(d_k(o_i, o_j) = d_k(o_j, o_i)\).

  2. Identité : \(d_k(o_i, o_i) = 0\).

  3. Inégalité triangulaire : \(d_k(o_i, o_j) \leq d_k(o_i, o_l) + d_k(o_l, o_j)\).

Preuves

Proof. Pour prouver que \(d_{GL}\) est une distance, nous devons vérifier les trois propriétés métriques.

1. **Symétrie** : Par construction, chaque \(d_k\) est symétrique. Ainsi,

\[d_{GL}(o_i, o_j) = \frac{\sum_{k=1}^{p} w_k d_k(o_i, o_j)}{\sum_{k=1}^{p} w_k} = \frac{\sum_{k=1}^{p} w_k d_k(o_j, o_i)}{\sum_{k=1}^{p} w_k} = d_{GL}(o_j, o_i).\]

2. **Identité** : Pour tout \(o_i \in \mathcal{O}\),

\[d_{GL}(o_i, o_i) = \frac{\sum_{k=1}^{p} w_k d_k(o_i, o_i)}{\sum_{k=1}^{p} w_k} = \frac{\sum_{k=1}^{p} w_k \cdot 0}{\sum_{k=1}^{p} w_k} = 0.\]

3. **Inégalité Triangulaire** : Supposons que \(d_k\) vérifie l’inégalité triangulaire. Alors,

\[d_{GL}(o_i, o_j) = \frac{\sum_{k=1}^{p} w_k d_k(o_i, o_j)}{\sum_{k=1}^{p} w_k} \leq \frac{\sum_{k=1}^{p} w_k (d_k(o_i, o_l) + d_k(o_l, o_j))}{\sum_{k=1}^{p} w_k} = d_{GL}(o_i, o_l) + d_{GL}(o_l, o_j).\]

Ainsi, \(d_{GL}\) est une distance. ◻

Propriétés et Corollaires

Nous listons maintenant quelques propriétés importantes de la distance de Gower-Legendre.

La distance de Gower-Legendre est invariante par transformation affine des variables quantitatives.

Proof. Soit \(\phi_k\) une transformation affine de la variable quantitative \(k\), c’est-à-dire \(\phi_k(x) = a_k x + b_k\). Alors,

\[d_k(o_i, o_j) = \frac{|\phi_k(o_{ik}) - \phi_k(o_{jk})|}{R_k} = \frac{|a_k (o_{ik} - o_{jk})|}{R_k} = |a_k| \frac{|o_{ik} - o_{jk}|}{R_k}.\]

Ainsi, la dissimilarité est multipliée par un facteur constant \(|a_k|\), ce qui ne change pas la distance globale \(d_{GL}\) car les poids \(w_k\) sont également multipliés par ce facteur. ◻

La distance de Gower-Legendre est appropriée pour l’analyse des données mixtes, car elle traite de manière cohérente les variables quantitatives, ordinales et nominales.

Proof. La définition de \(d_{GL}\) intègre explicitement les différentes natures des variables. Les variables quantitatives sont normalisées par leur étendue, les variables ordinales par le nombre de modalités, et les variables nominales par une dissimilarité binaire. Ainsi, \(d_{GL}\) capture la dissimilarité de manière adaptée à chaque type de variable. ◻

Conclusion

La distance de Gower-Legendre est un outil puissant pour l’analyse des données multidimensionnelles mixtes. Son élégance réside dans sa capacité à intégrer différentes natures de variables tout en préservant les propriétés métriques fondamentales. Elle trouve des applications dans divers domaines, de la biologie à la sociologie, en passant par l’apprentissage automatique.
