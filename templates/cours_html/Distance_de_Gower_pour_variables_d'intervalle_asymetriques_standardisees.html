Distance de Gower pour variables d’intervalle asymétriques standardisées

Quantix

2026-01-11

Introduction et Motivations

L’analyse des données multidimensionnelles nécessite souvent la définition d’une mesure de dissimilarité appropriée. La distance de Gower, introduite par le statisticien britannique Michael A. Gower en 1971, est une mesure de dissimilarité particulièrement adaptée aux données mixtes, c’est-à-dire contenant à la fois des variables quantitatives et qualitatives. Dans ce chapitre, nous nous concentrons sur l’extension de cette distance aux variables d’intervalle asymétriques standardisées.

Les variables d’intervalle asymétriques sont courantes dans de nombreux domaines, tels que la biologie, l’économie ou les sciences sociales. Elles se caractérisent par une distribution non symétrique autour de leur moyenne, ce qui rend les méthodes classiques de standardisation inappropriées. La distance de Gower, grâce à sa flexibilité, permet de prendre en compte cette asymétrie et d’obtenir une mesure de dissimilarité pertinente.

Définitions

Pour définir la distance de Gower pour des variables d’intervalle asymétriques standardisées, commençons par comprendre ce que nous cherchons à obtenir. Nous voulons une mesure de dissimilarité qui :

  • Prend en compte l’asymétrie des variables.

  • Standardise les variables de manière appropriée.

  • Est applicable à des données mixtes.

Nous cherchons donc une fonction \(d: \mathbb{R}^n \times \mathbb{R}^n \rightarrow [0,1]\) qui vérifie ces propriétés.

Soient \(\mathbf{x}, \mathbf{y} \in \mathbb{R}^n\) deux vecteurs de données, et \(X_1, X_2, \dots, X_n\) des variables d’intervalle asymétriques. Pour chaque variable \(X_j\), définissons une fonction de standardisation \(s_j: \mathbb{R} \rightarrow \mathbb{R}\) qui prend en compte l’asymétrie de \(X_j\).

La distance de Gower entre \(\mathbf{x}\) et \(\mathbf{y}\) est définie par : \[d(\mathbf{x}, \mathbf{y}) = \frac{\sum_{j=1}^n \delta_j(x_j, y_j)}{\sum_{j=1}^n u_j}\] où \(\delta_j(x_j, y_j)\) est la dissimilarité pour la variable \(X_j\), et \(u_j\) est un poids associé à la variable \(X_j\).

Pour une variable d’intervalle asymétrique standardisée, la dissimilarité est définie par : \[\delta_j(x_j, y_j) = \frac{|s_j(x_j) - s_j(y_j)|}{\max_{k,l} |s_j(X_{jk}) - s_j(X_{lk})|}\] où \(X_{jk}\) et \(X_{lk}\) sont les valeurs extrêmes de la variable \(X_j\).

Théorèmes

Pour établir les propriétés de la distance de Gower pour des variables d’intervalle asymétriques standardisées, nous avons besoin de quelques théorèmes fondamentaux.

La distance de Gower \(d\) définie ci-dessus vérifie les propriétés suivantes :

  1. Non-négativité : \(d(\mathbf{x}, \mathbf{y}) \geq 0\) pour tout \(\mathbf{x}, \mathbf{y} \in \mathbb{R}^n\).

  2. Identité : \(d(\mathbf{x}, \mathbf{y}) = 0\) si et seulement si \(\mathbf{x} = \mathbf{y}\).

  3. Symétrie : \(d(\mathbf{x}, \mathbf{y}) = d(\mathbf{y}, \mathbf{x})\) pour tout \(\mathbf{x}, \mathbf{y} \in \mathbb{R}^n\).

  4. Inégalité triangulaire : \(d(\mathbf{x}, \mathbf{z}) \leq d(\mathbf{x}, \mathbf{y}) + d(\mathbf{y}, \mathbf{z})\) pour tout \(\mathbf{x}, \mathbf{y}, \mathbf{z} \in \mathbb{R}^n\).

Preuves

Nous allons maintenant prouver les propriétés de la distance de Gower.

Preuve des propriétés de la distance de Gower.

  1. Non-négativité : Par définition, \(d(\mathbf{x}, \mathbf{y})\) est une somme de valeurs absolues, donc \(d(\mathbf{x}, \mathbf{y}) \geq 0\).

  2. Identité : Si \(\mathbf{x} = \mathbf{y}\), alors \(d(\mathbf{x}, \mathbf{y}) = 0\). Réciproquement, si \(d(\mathbf{x}, \mathbf{y}) = 0\), alors \(\delta_j(x_j, y_j) = 0\) pour tout \(j\), ce qui implique \(x_j = y_j\) pour tout \(j\).

  3. Symétrie : Par définition, \(\delta_j(x_j, y_j) = \delta_j(y_j, x_j)\), donc \(d(\mathbf{x}, \mathbf{y}) = d(\mathbf{y}, \mathbf{x})\).

  4. Inégalité triangulaire : Pour chaque \(j\), nous avons : \[\delta_j(x_j, z_j) \leq \delta_j(x_j, y_j) + \delta_j(y_j, z_j)\] En sommant sur \(j\) et en divisant par \(\sum_{j=1}^n u_j\), nous obtenons : \[d(\mathbf{x}, \mathbf{z}) \leq d(\mathbf{x}, \mathbf{y}) + d(\mathbf{y}, \mathbf{z})\]

Propriétés et Corollaires

Nous allons maintenant énoncer quelques propriétés et corollaires importants.

  1. La distance de Gower est invariante par transformation affine des variables.

  2. La distance de Gower peut être utilisée pour des données mixtes, c’est-à-dire contenant à la fois des variables quantitatives et qualitatives.

  3. La distance de Gower prend en compte l’asymétrie des variables d’intervalle.

Preuve des propriétés de la distance de Gower.

  1. Soit \(T_j: \mathbb{R} \rightarrow \mathbb{R}\) une transformation affine, \(T_j(x) = a_j x + b_j\). La dissimilarité devient : \[\delta_j(T_j(x_j), T_j(y_j)) = \frac{|s_j(T_j(x_j)) - s_j(T_j(y_j))|}{\max_{k,l} |s_j(T_j(X_{jk})) - s_j(T_j(X_{lk}))|}\] En utilisant les propriétés de \(s_j\), on montre que \(\delta_j(T_j(x_j), T_j(y_j)) = \delta_j(x_j, y_j)\).

  2. Pour des variables qualitatives, la dissimilarité peut être définie de manière appropriée, par exemple en utilisant une matrice de dissimilarité.

  3. La standardisation \(s_j\) prend en compte l’asymétrie de la variable \(X_j\), ce qui permet de définir une dissimilarité appropriée.

 ◻

La distance de Gower peut être utilisée pour effectuer une analyse de cluster ou une analyse des correspondances multiples sur des données mixtes.

Conclusion

Dans ce chapitre, nous avons défini et étudié la distance de Gower pour des variables d’intervalle asymétriques standardisées. Nous avons montré que cette distance vérifie les propriétés fondamentales d’une mesure de dissimilarité et peut être utilisée pour analyser des données mixtes. Les applications de cette distance sont nombreuses, notamment en biologie, en économie et en sciences sociales.
