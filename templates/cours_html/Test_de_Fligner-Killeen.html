Le Test de Fligner-Killeen: Une Approche Non-Paramétrique pour la Comparaison des Médianes

Quantix

2026-01-11

Introduction et Motivations

En statistique, la comparaison des médianes de plusieurs échantillons indépendants est une tâche fondamentale. Cependant, les méthodes classiques comme le test de Kruskal-Wallis supposent souvent que les distributions sous-jacentes ont des formes similaires. Le test de Fligner-Killeen, développé par Fligner et Killeen en 1976, offre une alternative non-paramétrique robuste à cette hypothèse restrictive.

L’intérêt pour ce test émerge naturellement lorsque l’on considère des données issues de distributions asymétriques ou multiformes. Par exemple, en biologie, les temps de survie peuvent suivre une distribution exponentielle, tandis que les concentrations de substances chimiques peuvent être log-normales. Dans de tels cas, les méthodes paramétriques traditionnelles peuvent conduire à des conclusions erronées.

Le test de Fligner-Killeen est particulièrement utile lorsqu’on souhaite comparer les médianes de plusieurs groupes tout en tenant compte des différences potentielles dans la forme des distributions. Il est basé sur une transformation des données qui permet de rendre les distributions plus symétriques, facilitant ainsi la comparaison des médianes.

Définitions

Avant de présenter le test de Fligner-Killeen, il est essentiel de comprendre quelques concepts prérequis.

Transformation de Rang

Considérons un échantillon de données \(X_1, X_2, \ldots, X_n\). La transformation de rang consiste à remplacer chaque observation \(X_i\) par son rang dans l’échantillon ordonné. Formellement, pour chaque \(i\), on définit le rang \(R_i\) comme suit :

\[R_i = \sum_{j=1}^n \mathbb{I}(X_j \leq X_i)\]

où \(\mathbb{I}\) est l’indicatrice qui vaut 1 si la condition est vraie et 0 sinon.

Test de Fligner-Killeen

Le test de Fligner-Killeen est une extension du test de Kruskal-Wallis qui prend en compte les différences dans la forme des distributions. Il est basé sur une statistique de test \(T\) définie comme suit :

\[T = \sum_{i=1}^k \frac{n_i}{n} \left( \frac{R_i - \mu_i}{\sigma_i} \right)^2\]

où : - \(k\) est le nombre de groupes, - \(n_i\) est la taille de l’échantillon du groupe \(i\), - \(n\) est la taille totale de l’échantillon, - \(R_i\) est la somme des rangs pour le groupe \(i\), - \(\mu_i\) et \(\sigma_i\) sont la moyenne et l’écart-type des rangs pour le groupe \(i\).

Théorèmes

Théorème de Fligner-Killeen

Le théorème central du test de Fligner-Killeen stipule que sous l’hypothèse nulle d’égalité des médianes, la statistique de test \(T\) suit une distribution du chi-deux à \(k-1\) degrés de liberté.

Soient \(X_{11}, \ldots, X_{1n_1}, \ldots, X_{k1}, \ldots, X_{kn_k}\) des échantillons indépendants issus de distributions avec des médianes \(m_1, \ldots, m_k\). Sous l’hypothèse nulle \(H_0: m_1 = \ldots = m_k\), la statistique de test

\[T = \sum_{i=1}^k \frac{n_i}{n} \left( \frac{R_i - \mu_i}{\sigma_i} \right)^2\]

suit une loi du chi-deux à \(k-1\) degrés de liberté, où \(R_i\), \(\mu_i\) et \(\sigma_i\) sont définis comme précédemment.

Preuves

Preuve du Théorème de Fligner-Killeen

La preuve du théorème de Fligner-Killeen repose sur plusieurs étapes clés.

Proof. 1. **Transformation des Données** : Tout d’abord, nous transformons les données en utilisant la transformation de rang. Cela permet de rendre les distributions plus symétriques et facilite la comparaison des médianes.

2. **Calcul des Rangs** : Pour chaque groupe \(i\), nous calculons la somme des rangs \(R_i\). Cette étape est cruciale car elle permet de prendre en compte les différences dans la forme des distributions.

3. **Normalisation** : Nous normalisons les sommes de rangs en soustrayant la moyenne \(\mu_i\) et en divisant par l’écart-type \(\sigma_i\). Cela permet de rendre les statistiques comparables entre les groupes.

4. **Statistique de Test** : Nous calculons la statistique de test \(T\) en utilisant la formule donnée dans le théorème. Sous l’hypothèse nulle, cette statistique suit une distribution du chi-deux à \(k-1\) degrés de liberté.

5. **Conclusion** : En comparant la valeur calculée de \(T\) à la distribution du chi-deux, nous pouvons rejeter ou ne pas rejeter l’hypothèse nulle d’égalité des médianes. ◻

Propriétés et Corollaires

Propriétés du Test de Fligner-Killeen

Le test de Fligner-Killeen possède plusieurs propriétés importantes :

  1. **Robustesse** : Le test est robuste aux différences dans la forme des distributions, ce qui le rend particulièrement utile pour les données asymétriques ou multiformes.

  2. **Non-Paramétrique** : Le test ne suppose pas de forme particulière pour les distributions sous-jacentes, ce qui le rend applicable à une large gamme de données.

  3. **Puissance** : Le test est puissant pour détecter les différences dans les médianes, même lorsque les distributions sont asymétriques.

Corollaires

Le test de Fligner-Killeen peut être utilisé pour dériver plusieurs corollaires intéressants. Par exemple, il peut être utilisé pour comparer les médianes de plus de deux groupes, ce qui est une extension directe du test de Kruskal-Wallis.

Conclusion

Le test de Fligner-Killeen est une méthode non-paramétrique puissante pour la comparaison des médianes de plusieurs échantillons indépendants. Il offre une alternative robuste aux méthodes traditionnelles comme le test de Kruskal-Wallis, en tenant compte des différences potentielles dans la forme des distributions. Son utilisation est particulièrement recommandée lorsque les données sont asymétriques ou multiformes.
