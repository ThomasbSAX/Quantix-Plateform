Dérivées des fonctions trigonométriques

Quantix

2026-01-11

Introduction et Motivations

Les fonctions trigonométriques, héritées des travaux de mathématiciens grecs anciens comme Hipparque et Ptolémée, ont évolué pour devenir des outils fondamentaux en analyse mathématique. Leur dérivabilité, étudiée par Newton et Leibniz au XVIIe siècle, est cruciale pour résoudre des équations différentielles, modéliser des phénomènes périodiques et analyser les oscillations. Ces dérivées permettent de comprendre le taux de variation instantané des fonctions trigonométriques, essentiel en physique pour décrire les mouvements harmoniques ou les ondes.

Définitions

Pour définir les dérivées des fonctions trigonométriques, nous partons du principe que ces fonctions sont continues et différentiables sur \(\mathbb{R}\). Nous cherchons à déterminer la dérivée de \(f(x) = \sin x\) et \(g(x) = \cos x\).

Soit \(f(x) = \sin x\). Nous voulons trouver une fonction \(f'(x)\) telle que : \[\lim_{h \to 0} \frac{\sin(x + h) - \sin x}{h}\] En utilisant l’identité trigonométrique \(\sin(x + h) = \sin x \cos h + \cos x \sin h\), nous obtenons : \[f'(x) = \lim_{h \to 0} \frac{\sin x (\cos h - 1) + \cos x \sin h}{h}\] En séparant les termes et en utilisant les limites connues \(\lim_{h \to 0} \frac{\sin h}{h} = 1\) et \(\lim_{h \to 0} \frac{\cos h - 1}{h} = 0\), nous trouvons : \[f'(x) = \cos x\]

Soit \(g(x) = \cos x\). Nous voulons trouver une fonction \(g'(x)\) telle que : \[\lim_{h \to 0} \frac{\cos(x + h) - \cos x}{h}\] En utilisant l’identité trigonométrique \(\cos(x + h) = \cos x \cos h - \sin x \sin h\), nous obtenons : \[g'(x) = \lim_{h \to 0} \frac{\cos x (\cos h - 1) - \sin x \sin h}{h}\] En séparant les termes et en utilisant les mêmes limites que précédemment, nous trouvons : \[g'(x) = -\sin x\]

Théorèmes

Nous présentons maintenant quelques théorèmes importants concernant les dérivées des fonctions trigonométriques.

Soit \(h(x) = \tan x\). Nous voulons trouver une fonction \(h'(x)\) telle que : \[\lim_{k \to 0} \frac{\tan(x + k) - \tan x}{k}\] En utilisant l’identité trigonométrique \(\tan(x + k) = \frac{\tan x + \tan k}{1 - \tan x \tan k}\), nous obtenons : \[h'(x) = \lim_{k \to 0} \frac{\frac{\tan x + \tan k}{1 - \tan x \tan k} - \tan x}{k}\] En simplifiant l’expression, nous trouvons : \[h'(x) = \lim_{k \to 0} \frac{\tan k}{k(1 - \tan x \tan k)} = \frac{1}{1 + \tan^2 x} = \cos^2 x\]

Soit \(m(x) = \cot x\). Nous voulons trouver une fonction \(m'(x)\) telle que : \[\lim_{l \to 0} \frac{\cot(x + l) - \cot x}{l}\] En utilisant l’identité trigonométrique \(\cot(x + l) = \frac{\cot x \cot l - 1}{\cot l - \cot x}\), nous obtenons : \[m'(x) = \lim_{l \to 0} \frac{\frac{\cot x \cot l - 1}{\cot l - \cot x} - \cot x}{l}\] En simplifiant l’expression, nous trouvons : \[m'(x) = \lim_{l \to 0} \frac{-1}{l(\cot l - \cot x)} = -\frac{1}{\sin^2 x}\]

Preuves

Nous fournissons maintenant les preuves détaillées des théorèmes présentés.

Preuve du théorème sur la dérivée de la fonction tangente. Nous avons : \[h'(x) = \lim_{k \to 0} \frac{\tan(x + k) - \tan x}{k}\] En utilisant l’identité trigonométrique pour \(\tan(x + k)\), nous obtenons : \[h'(x) = \lim_{k \to 0} \frac{\frac{\tan x + \tan k}{1 - \tan x \tan k} - \tan x}{k}\] En simplifiant le numérateur : \[h'(x) = \lim_{k \to 0} \frac{\tan x + \tan k - \tan x (1 - \tan x \tan k)}{k(1 - \tan x \tan k)} = \lim_{k \to 0} \frac{\tan k + \tan^2 x \tan k}{k(1 - \tan x \tan k)}\] En factorisant \(\tan k\) : \[h'(x) = \lim_{k \to 0} \frac{\tan k (1 + \tan^2 x)}{k(1 - \tan x \tan k)} = (1 + \tan^2 x) \lim_{k \to 0} \frac{\tan k}{k}\] En utilisant la limite connue \(\lim_{k \to 0} \frac{\tan k}{k} = 1\), nous obtenons : \[h'(x) = (1 + \tan^2 x) \cdot 1 = 1 + \tan^2 x\] En utilisant l’identité trigonométrique \(1 + \tan^2 x = \sec^2 x\), nous avons : \[h'(x) = \sec^2 x = \frac{1}{\cos^2 x}\] ◻

Preuve du théorème sur la dérivée de la fonction cotangente. Nous avons : \[m'(x) = \lim_{l \to 0} \frac{\cot(x + l) - \cot x}{l}\] En utilisant l’identité trigonométrique pour \(\cot(x + l)\), nous obtenons : \[m'(x) = \lim_{l \to 0} \frac{\frac{\cot x \cot l - 1}{\cot l - \cot x} - \cot x}{l}\] En simplifiant le numérateur : \[m'(x) = \lim_{l \to 0} \frac{\cot x \cot l - 1 - \cot x (\cot l - \cot x)}{l(\cot l - \cot x)} = \lim_{l \to 0} \frac{\cot x \cot l - 1 - \cot x \cot l + \cot^2 x}{l(\cot l - \cot x)}\] En simplifiant le numérateur : \[m'(x) = \lim_{l \to 0} \frac{-1 + \cot^2 x}{l(\cot l - \cot x)} = \lim_{l \to 0} \frac{\cot^2 x - 1}{l(\cot l - \cot x)}\] En utilisant l’identité trigonométrique \(\cot^2 x - 1 = \frac{\cos^2 x}{\sin^2 x} - 1 = \frac{\cos^2 x - \sin^2 x}{\sin^2 x} = \frac{\cos(2x)}{\sin^2 x}\), nous obtenons : \[m'(x) = \lim_{l \to 0} \frac{\cos(2x)}{l \sin^2 x (\cot l - \cot x)}\] En utilisant la limite connue \(\lim_{l \to 0} (\cot l - \cot x) = -\csc^2 x \cdot l\), nous avons : \[m'(x) = \lim_{l \to 0} \frac{\cos(2x)}{l \sin^2 x (-\csc^2 x \cdot l)} = -\frac{\cos(2x)}{\sin^2 x}\] En utilisant l’identité trigonométrique \(\cos(2x) = 1 - 2\sin^2 x\), nous obtenons : \[m'(x) = -\frac{1 - 2\sin^2 x}{\sin^2 x} = -\frac{1}{\sin^2 x}\] ◻

Propriétés et Corollaires

Nous présentons maintenant quelques propriétés et corollaires des dérivées des fonctions trigonométriques.

Soit \(n(x) = \sec x\) et \(p(x) = \csc x\). Nous avons : \[n'(x) = \frac{d}{dx} (\sec x) = \sec x \tan x\] \[p'(x) = \frac{d}{dx} (\csc x) = -\csc x \cot x\]

Preuve du corollaire sur les dérivées des fonctions sécante et cosécante. Pour \(n(x) = \sec x\), nous avons : \[n'(x) = \frac{d}{dx} (\sec x) = \frac{d}{dx} \left( \frac{1}{\cos x} \right) = -\frac{-\sin x}{\cos^2 x} = \sec x \tan x\] Pour \(p(x) = \csc x\), nous avons : \[p'(x) = \frac{d}{dx} (\csc x) = \frac{d}{dx} \left( \frac{1}{\sin x} \right) = -\frac{\cos x}{\sin^2 x} = -\csc x \cot x\] ◻

Soit \(q(x) = \arcsin x\), \(r(x) = \arccos x\) et \(s(x) = \arctan x\). Nous avons : \[q'(x) = \frac{1}{\sqrt{1 - x^2}}\] \[r'(x) = -\frac{1}{\sqrt{1 - x^2}}\] \[s'(x) = \frac{1}{1 + x^2}\]

Preuve du corollaire sur les dérivées des fonctions trigonométriques inverses. Pour \(q(x) = \arcsin x\), nous avons : \[q'(x) = \frac{d}{dx} (\arcsin x) = \frac{1}{\sqrt{1 - x^2}}\] Pour \(r(x) = \arccos x\), nous avons : \[r'(x) = \frac{d}{dx} (\arccos x) = -\frac{1}{\sqrt{1 - x^2}}\] Pour \(s(x) = \arctan x\), nous avons : \[s'(x) = \frac{d}{dx} (\arctan x) = \frac{1}{1 + x^2}\] ◻
