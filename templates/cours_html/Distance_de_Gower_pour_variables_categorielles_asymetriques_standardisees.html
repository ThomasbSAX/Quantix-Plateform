Distance de Gower pour variables catégorielles asymétriques standardisées

Introduction et Motivations

L’analyse des données catégorielles est un champ fondamental en statistique et en apprentissage automatique. Les variables catégorielles, souvent qualitatives, nécessitent des outils spécifiques pour mesurer leurs similarités. La distance de Gower émerge comme une solution élégante pour comparer des objets décrits par des variables mixtes, combinant à la fois des variables quantitatives et qualitatives.

Historiquement, la distance de Gower a été introduite par G. A. C. Gower en 1971 pour répondre au besoin de mesurer la dissimilarité entre des entités décrites par des variables de types différents. Son approche innovante permet de standardiser les variables, rendant la mesure robuste aux différences d’échelle et aux asymétries dans les données catégorielles.

Dans ce chapitre, nous explorerons la distance de Gower pour des variables catégorielles asymétriques standardisées. Nous verrons comment cette distance résout les problèmes de comparaison entre variables qualitatives et quantifie leur dissimilarité de manière cohérente.

Définitions

Pour définir la distance de Gower, commençons par comprendre ce que nous cherchons à mesurer. Nous voulons quantifier la dissimilarité entre deux objets décrits par des variables catégorielles. Pour cela, nous devons tenir compte de la nature qualitative des variables et de leurs éventuelles asymétries.

Supposons que nous avons deux objets \(x\) et \(y\) décrits par un ensemble de variables catégorielles \(V = \{v_1, v_2, \ldots, v_p\}\). Pour chaque variable \(v_j\), nous voulons mesurer la dissimilarité entre les valeurs \(x_j\) et \(y_j\).

Formellement, la distance de Gower pour des variables catégorielles peut être définie comme suit :

Soient \(x\) et \(y\) deux objets décrits par des variables catégorielles \(V = \{v_1, v_2, \ldots, v_p\}\). La distance de Gower \(d_G(x, y)\) est définie par : \[d_G(x, y) = \frac{1}{p} \sum_{j=1}^p d_j(x_j, y_j)\] où \(d_j(x_j, y_j)\) est la dissimilarité entre les valeurs \(x_j\) et \(y_j\) pour la variable \(v_j\).

Pour les variables catégorielles, la dissimilarité \(d_j(x_j, y_j)\) peut être définie comme suit :

Soient \(x_j\) et \(y_j\) les valeurs de la variable catégorielle \(v_j\) pour les objets \(x\) et \(y\). La dissimilarité \(d_j(x_j, y_j)\) est définie par : \[d_j(x_j, y_j) = \begin{cases} 0 & \text{si } x_j = y_j, \\ 1 & \text{sinon.} \end{cases}\]

Cependant, cette définition ne tient pas compte des asymétries dans les variables catégorielles. Pour standardiser ces variables, nous introduisons une version pondérée de la distance de Gower.

Théorèmes

Pour mieux comprendre les propriétés de la distance de Gower, nous allons explorer quelques théorèmes fondamentaux. Supposons que nous avons une matrice de dissimilarité \(D\) où chaque élément \(D_{xy}\) représente la distance entre les objets \(x\) et \(y\).

La distance de Gower \(d_G\) satisfait les propriétés suivantes :

  1. Non-négativité : \(d_G(x, y) \geq 0\) pour tout \(x, y\).

  2. Identité des indiscernables : \(d_G(x, y) = 0\) si et seulement si \(x = y\).

  3. Symétrie : \(d_G(x, y) = d_G(y, x)\) pour tout \(x, y\).

  4. Inégalité triangulaire : \(d_G(x, z) \leq d_G(x, y) + d_G(y, z)\) pour tout \(x, y, z\).

Proof. Nous allons démontrer chaque propriété une par une.

1. Non-négativité : Puisque \(d_j(x_j, y_j) \geq 0\) pour tout \(j\), il s’ensuit que : \[d_G(x, y) = \frac{1}{p} \sum_{j=1}^p d_j(x_j, y_j) \geq 0.\]

2. Identité des indiscernables : Si \(x = y\), alors \(d_j(x_j, y_j) = 0\) pour tout \(j\), donc : \[d_G(x, y) = 0.\] Réciproquement, si \(d_G(x, y) = 0\), alors \(d_j(x_j, y_j) = 0\) pour tout \(j\), ce qui implique que \(x_j = y_j\) pour tout \(j\). Par conséquent, \(x = y\).

3. Symétrie : Puisque \(d_j(x_j, y_j) = d_j(y_j, x_j)\) pour tout \(j\), nous avons : \[d_G(x, y) = \frac{1}{p} \sum_{j=1}^p d_j(x_j, y_j) = \frac{1}{p} \sum_{j=1}^p d_j(y_j, x_j) = d_G(y, x).\]

4. Inégalité triangulaire : Pour tout \(j\), nous avons : \[d_j(x_j, z_j) \leq d_j(x_j, y_j) + d_j(y_j, z_j).\] En sommant sur \(j\) et en divisant par \(p\), nous obtenons : \[d_G(x, z) = \frac{1}{p} \sum_{j=1}^p d_j(x_j, z_j) \leq \frac{1}{p} \sum_{j=1}^p (d_j(x_j, y_j) + d_j(y_j, z_j)) = d_G(x, y) + d_G(y, z).\] ◻

Preuves

Pour illustrer les preuves détaillées, considérons un exemple simple. Supposons que nous avons deux objets \(x\) et \(y\) décrits par trois variables catégorielles \(v_1, v_2, v_3\). Les valeurs des variables sont les suivantes : \[x = (a, b, c), \quad y = (a, d, e).\]

La dissimilarité pour chaque variable est : \[d_1(x_1, y_1) = 0 \quad (\text{car } x_1 = y_1), \quad d_2(x_2, y_2) = 1 \quad (\text{car } x_2 \neq y_2), \quad d_3(x_3, y_3) = 1 \quad (\text{car } x_3 \neq y_3).\]

La distance de Gower est alors : \[d_G(x, y) = \frac{1}{3} (0 + 1 + 1) = \frac{2}{3}.\]

Propriétés et Corollaires

Nous allons maintenant explorer quelques propriétés et corollaires de la distance de Gower.

La distance de Gower peut être standardisée pour tenir compte des asymétries dans les variables catégorielles. Pour chaque variable \(v_j\), nous pouvons introduire un poids \(w_j\) tel que : \[d_G(x, y) = \frac{1}{\sum_{j=1}^p w_j} \sum_{j=1}^p w_j d_j(x_j, y_j).\]

Proof. La standardisation des variables permet de pondérer les dissimilarités en fonction de l’importance relative de chaque variable. En introduisant des poids \(w_j\), nous pouvons ajuster la contribution de chaque variable à la distance globale. ◻

La distance de Gower est invariante par translation des variables catégorielles. Cela signifie que si nous ajoutons une constante à toutes les valeurs d’une variable, la distance de Gower reste inchangée.

Proof. Puisque les variables catégorielles sont qualitatives, l’ajout d’une constante à leurs valeurs ne change pas leur nature. Par conséquent, la dissimilarité entre deux objets reste inchangée. ◻

Conclusion

La distance de Gower pour variables catégorielles asymétriques standardisées est un outil puissant pour mesurer la dissimilarité entre des objets décrits par des variables mixtes. Ses propriétés fondamentales, telles que la non-négativité, l’identité des indiscernables, la symétrie et l’inégalité triangulaire, en font une mesure robuste et cohérente. En standardisant les variables et en introduisant des poids, nous pouvons tenir compte des asymétries et de l’importance relative de chaque variable.

Ce chapitre a exploré les définitions, théorèmes et propriétés de la distance de Gower, fournissant une base solide pour son utilisation dans l’analyse des données catégorielles.
