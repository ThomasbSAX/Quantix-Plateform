Décomposition ANOVA à deux facteurs

Quantix

2026-01-11

Introduction et Motivations

L’analyse de la variance (ANOVA) est une méthode statistique fondamentale pour comparer les moyennes de plusieurs groupes. La décomposition ANOVA à deux facteurs permet d’étudier l’effet de deux variables catégorielles sur une variable réponse. Cette approche est indispensable dans les sciences expérimentales pour identifier les sources de variation et comprendre leurs interactions.

Définitions

Considérons un modèle linéaire général à deux facteurs avec interactions : \[Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \epsilon_{ijk}\] où :

  • \(Y_{ijk}\) est la réponse pour le niveau \(i\) du facteur A, le niveau \(j\) du facteur B et la réplique \(k\).

  • \(\mu\) est la moyenne globale.

  • \(\alpha_i\) est l’effet du niveau \(i\) du facteur A.

  • \(\beta_j\) est l’effet du niveau \(j\) du facteur B. \[\alpha_i = \mu_Ai - \mu, \quad \beta_j = \mu_Bj - \mu\] où \(\mu_Ai\) et \(\mu_Bj\) sont les moyennes marginales.

  • \((\alpha\beta)_{ij}\) est l’effet d’interaction entre les niveaux \(i\) et \(j\). \[(\alpha\beta)_{ij} = \mu_{ABij} - \mu_Ai - \mu_Bj + \mu\] où \(\mu_{ABij}\) est la moyenne pour le couple \((i,j)\).

  • \(\epsilon_{ijk}\) est l’erreur aléatoire, supposée normale et indépendante.

Théorèmes

La somme totale des carrés (SST) peut être décomposée en : \[SST = SSC_A + SSC_B + SSC_{AB} + SSE\] où :

  • \(SSC_A = n \sum_{i=1}^{a} (\bar{Y}_{i..} - \bar{Y}_{...})^2\) est la somme des carrés pour le facteur A.

  • \(SSC_B = bn \sum_{j=1}^{b} (\bar{Y}_{.j.} - \bar{Y}_{...})^2\) est la somme des carrés pour le facteur B.

  • \(SSC_{AB} = n \sum_{i=1}^{a} \sum_{j=1}^{b} (\bar{Y}_{ij.} - \bar{Y}_{i..} - \bar{Y}_{.j.} + \bar{Y}_{...})^2\) est la somme des carrés pour l’interaction.

  • \(SSE = \sum_{i=1}^{a} \sum_{j=1}^{b} \sum_{k=1}^{n} (Y_{ijk} - \bar{Y}_{ij.})^2\) est la somme des carrés des erreurs.

Preuves

Preuve du théorème de décomposition. La preuve repose sur l’orthogonalité des effets dans le modèle ANOVA. Nous avons : \[SST = \sum_{i=1}^{a} \sum_{j=1}^{b} \sum_{k=1}^{n} (Y_{ijk} - \bar{Y}_{...})^2\] En utilisant les identités suivantes : \[Y_{ijk} - \bar{Y}_{...} = (Y_{ijk} - \bar{Y}_{ij.}) + (\bar{Y}_{ij.} - \bar{Y}_{i..}) + (\bar{Y}_{i..} - \bar{Y}_{...})\] et en développant les carrés, nous obtenons la décomposition souhaitée. ◻

Propriétés et Corollaires

Les hypothèses principales sont :

  1. Les effets des facteurs et leurs interactions sont additifs. \[Y_{ijk} = \mu + \alpha_i + \beta_j + (\alpha\beta)_{ij} + \epsilon_{ijk}\]

  2. Les erreurs \(\epsilon_{ijk}\) sont indépendantes et normalement distribuées avec moyenne zéro et variance constante. \[\epsilon_{ijk} \sim N(0, \sigma^2)\]

Preuve des hypothèses. L’additivité des effets est une conséquence du modèle linéaire général. La normalité et l’homogénéité des variances sont des hypothèses standard en ANOVA, nécessaires pour la validité des tests statistiques. ◻

Les tests F pour les effets principaux et l’interaction sont donnés par : \[F_A = \frac{SSC_A / (a-1)}{SSE / (ab(n-1))}, \quad F_B = \frac{SSC_B / (b-1)}{SSE / (ab(n-1))}, \quad F_{AB} = \frac{SSC_{AB} / ((a-1)(b-1))}{SSE / (ab(n-1))}\]

Preuve des tests F. Ces tests sont basés sur le rapport des carrés moyens. Sous les hypothèses nulles, ces rapports suivent une loi F de Satterthwaite. ◻

Conclusion

La décomposition ANOVA à deux facteurs est un outil puissant pour analyser les effets de plusieurs variables catégorielles sur une réponse. Elle permet de comprendre les interactions complexes et d’identifier les sources de variation dans les données expérimentales.
