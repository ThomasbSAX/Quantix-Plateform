Le Test de Breusch-Pagan : Détection d’Hétéroscédasticité dans les Modèles de Régression

Quantix

2026-01-11

Introduction et Motivations

L’analyse des données économiques et sociales repose souvent sur l’utilisation de modèles de régression linéaire. Un postulat fondamental de ces modèles est l’homoscédasticité, c’est-à-dire la constance de la variance des résidus. Cependant, dans de nombreuses applications empiriques, cette hypothèse est violée, conduisant à une hétéroscédasticité. L’hétéroscédasticité peut entraîner une inefficacité des estimateurs et fausser les tests d’hypothèses. Le test de Breusch-Pagan, développé par Trevor S. Breusch et Adrian R. Pagan en 1979, offre un moyen statistique rigoureux pour détecter la présence d’hétéroscédasticité dans les modèles de régression.

Ce test est particulièrement utile dans les contextes où les résidus semblent présenter une variance non constante. Par exemple, dans l’analyse des salaires, la variance des résidus pourrait augmenter avec le niveau d’éducation ou l’expérience professionnelle. Le test de Breusch-Pagan permet aux chercheurs de vérifier formellement cette hypothèse et d’ajuster leurs modèles en conséquence.

Définitions

Pour comprendre le test de Breusch-Pagan, il est essentiel de définir quelques concepts clés.

Modèle de Régression Linéaire

Considérons un modèle de régression linéaire classique : \[y_i = x_i' \beta + u_i, \quad i = 1, \ldots, n\] où \(y_i\) est la variable dépendante, \(x_i\) est un vecteur de variables explicatives, \(\beta\) est le vecteur des coefficients à estimer, et \(u_i\) est le terme d’erreur.

Hétéroscédasticité

L’hétéroscédasticité se produit lorsque la variance des termes d’erreur \(u_i\) n’est pas constante. Formellement, on dit qu’il y a hétéroscédasticité si : \[\text{Var}(u_i | x_i) = \sigma_i^2 \neq \sigma^2 \quad \text{pour certains } i\]

Test de Breusch-Pagan

Le test de Breusch-Pagan est un test statistique utilisé pour détecter l’hétéroscédasticité dans les modèles de régression linéaire. Il repose sur une régression aux moindres carrés des résidus au carré sur les variables explicatives et leurs carrés.

Théorèmes

Formulation du Test de Breusch-Pagan

Le test de Breusch-Pagan peut être formulé comme suit :

Supposons que nous avons estimé un modèle de régression linéaire par la méthode des moindres carrés ordinaires (MCO) et obtenu les résidus \(\hat{u}_i\). Nous voulons tester l’hypothèse nulle d’homoscédasticité contre une alternative d’hétéroscédasticité.

Le test de Breusch-Pagan consiste à effectuer la régression suivante : \[\hat{u}_i^2 = \gamma_0 + \gamma_1 x_i + \epsilon_i\] où \(\hat{u}_i^2\) sont les résidus au carré, \(x_i\) sont les variables explicatives, et \(\epsilon_i\) est le nouveau terme d’erreur.

La statistique de test est donnée par : \[BP = n R^2\] où \(n\) est le nombre d’observations et \(R^2\) est le coefficient de détermination de la régression auxiliaire.

Sous l’hypothèse nulle d’homoscédasticité, la statistique \(BP\) suit une loi du chi-deux avec \(k\) degrés de liberté, où \(k\) est le nombre de variables explicatives dans la régression auxiliaire.

Preuves

Démonstration de la Statistique de Test

Pour démontrer que la statistique \(BP\) suit une loi du chi-deux sous l’hypothèse nulle, nous procédons comme suit :

1. **Régression Auxiliaire** : Considérons la régression auxiliaire des résidus au carré sur les variables explicatives : \[\hat{u}_i^2 = \gamma_0 + \gamma_1 x_i + \epsilon_i\]

2. **Coefficient de Détermination** : Le coefficient de détermination \(R^2\) de cette régression est donné par : \[R^2 = \frac{\text{SSE}}{\text{SST}}\] où \(\text{SSE}\) est la somme des carrés expliqués et \(\text{SST}\) est la somme totale des carrés.

3. **Statistique de Test** : La statistique de test est définie comme : \[BP = n R^2\]

4. **Distribution sous l’Hypothèse Nulle** : Sous l’hypothèse nulle d’homoscédasticité, les résidus \(\hat{u}_i\) sont indépendants et identiquement distribués (i.i.d.) avec une variance constante. Il peut être montré que, sous cette hypothèse, \(BP\) suit une loi du chi-deux avec \(k\) degrés de liberté.

Propriétés et Corollaires

Propriétés du Test de Breusch-Pagan

Le test de Breusch-Pagan possède plusieurs propriétés importantes :

(i) **Puissance** : Le test est puissant contre les alternatives d’hétéroscédasticité, c’est-à-dire qu’il a une forte probabilité de rejeter l’hypothèse nulle lorsque celle-ci est fausse.

(ii) **Robustesse** : Le test est robuste aux violations de certaines hypothèses du modèle de régression linéaire, telles que la normalité des termes d’erreur.

(iii) **Simplicité** : Le test est relativement simple à mettre en œuvre et ne nécessite pas de calculs complexes.

Corollaires

Le test de Breusch-Pagan peut être étendu pour détecter des formes spécifiques d’hétéroscédasticité. Par exemple, si l’on suspecte que la variance des résidus dépend d’une variable spécifique \(z_i\), on peut effectuer une régression auxiliaire des résidus au carré sur \(z_i\) et utiliser la statistique \(BP\) pour tester l’hypothèse nulle.

Conclusion

Le test de Breusch-Pagan est un outil essentiel pour détecter l’hétéroscédasticité dans les modèles de régression linéaire. Sa simplicité et sa puissance en font un choix populaire parmi les chercheurs en économie et en sciences sociales. En comprenant les principes sous-jacents à ce test, les analystes peuvent mieux évaluer la validité de leurs modèles et prendre des décisions éclairées pour l’ajustement des données.
