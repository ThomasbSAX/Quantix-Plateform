Test de Levene : Une Analyse de l’Homogénéité des Variances

Quantix

2026-01-11

Introduction et Motivations

Le test de Levene, introduit par le statisticien Horace Levene en 1960, est un outil fondamental dans l’analyse des données. Il évalue l’homogénéité des variances entre plusieurs groupes, une condition cruciale pour de nombreuses techniques statistiques comme l’analyse de variance (ANOVA). L’importance de ce test réside dans sa capacité à détecter les violations de l’hypothèse d’homogénéité des variances, ce qui permet aux chercheurs de choisir les méthodes appropriées pour leurs analyses.

Définitions

Avant de formuler le test de Levene, il est essentiel de comprendre les concepts sous-jacents. Supposons que nous ayons \(k\) groupes de données, avec \(n_i\) observations dans le groupe \(i\), pour \(i = 1, 2, \dots, k\). Nous cherchons à tester si les variances de ces groupes sont égales.

Pour ce faire, nous définissons d’abord les résidus pour chaque observation \(X_{ij}\) par rapport à la médiane de son groupe : \[Z_{ij} = |X_{ij} - \text{Med}(X_i)|\] où \(\text{Med}(X_i)\) est la médiane du groupe \(i\).

Le test de Levene repose sur une régression des résidus \(Z_{ij}\) sur les indicateurs de groupe. Plus formellement, nous modélisons : \[Z_{ij} = \beta_0 + \beta_i I_i + \epsilon_{ij}\] où \(I_i\) est un indicateur égal à 1 si l’observation appartient au groupe \(i\) et 0 sinon, et \(\epsilon_{ij}\) est le terme d’erreur.

Théorèmes

Le test de Levene peut être formulé comme un test d’hypothèse. Nous voulons tester : \[H_0 : \sigma_1^2 = \sigma_2^2 = \dots = \sigma_k^2\] contre l’hypothèse alternative : \[H_1 : \text{Au moins deux variances sont différentes}\]

Pour ce faire, nous utilisons une ANOVA sur les résidus \(Z_{ij}\). Le statisticien Horace Levene a montré que sous l’hypothèse nulle, le ratio des variances entre-groupes et dans les groupes suit une distribution \(F\) avec \(k-1\) et \(N-k\) degrés de liberté, où \(N = \sum_{i=1}^k n_i\).

Preuves

Pour démontrer la validité du test de Levene, nous devons montrer que sous \(H_0\), le statisticien \(F\) suit bien une distribution \(F\). Commençons par calculer la somme des carrés entre-groupes : \[SSB = \sum_{i=1}^k n_i (\bar{Z}_i - \bar{Z})^2\] où \(\bar{Z}_i\) est la moyenne des résidus pour le groupe \(i\), et \(\bar{Z}\) est la moyenne globale des résidus.

Ensuite, calculons la somme des carrés dans les groupes : \[SSW = \sum_{i=1}^k \sum_{j=1}^{n_i} (Z_{ij} - \bar{Z}_i)^2\]

Le statisticien \(F\) est alors donné par : \[F = \frac{SSB / (k-1)}{SSW / (N-k)}\]

Sous \(H_0\), les résidus \(Z_{ij}\) sont indépendants et identiquement distribués, ce qui implique que \(SSB\) et \(SSW\) sont indépendants. De plus, \(SSB / (k-1)\) et \(SSW / (N-k)\) suivent des distributions du chi-carré, ce qui permet de conclure que \(F\) suit une distribution \(F\).

Propriétés et Corollaires

Le test de Levene possède plusieurs propriétés importantes :

(i) **Robustesse** : Le test de Levene est robuste aux violations de la normalité, ce qui le rend particulièrement utile pour les données non normales.

(ii) **Puissance** : Le test est puissant pour détecter les différences de variances, surtout lorsque les tailles des groupes sont égales.

(iii) **Alternatives** : Il existe plusieurs variantes du test de Levene, comme le test de Brown-Forsythe qui utilise la moyenne au lieu de la médiane. Ces variantes peuvent être plus appropriées dans certains contextes.

Conclusion

Le test de Levene est un outil indispensable pour évaluer l’homogénéité des variances. Sa simplicité et sa robustesse en font un choix privilégié pour de nombreuses applications statistiques. En comprenant ses fondements théoriques et ses propriétés, les chercheurs peuvent mieux interpréter leurs résultats et choisir les méthodes d’analyse appropriées.
