Distance de Gower pour variables discrètes asymétriques

Introduction et Motivations

L’analyse des données discrètes asymétriques est un enjeu majeur dans de nombreux domaines scientifiques, allant de la biologie à l’économie. La distance de Gower émerge comme une solution élégante pour mesurer la dissimilarité entre objets décrits par des variables mixtes, combinant à la fois des variables qualitatives et quantitatives. Cette distance est particulièrement adaptée aux contextes où les variables discrètes ne sont pas symétriques, c’est-à-dire que la dissimilarité entre deux modalités n’est pas nécessairement la même dans les deux sens.

L’origine de cette notion remonte aux travaux de Gower (1971), qui a introduit une mesure de dissimilarité générale pour les données mixtes. Cette approche est indispensable dans les analyses multivariées, où il est crucial de capturer la complexité des données sans perdre d’information. La distance de Gower permet de traiter les variables discrètes asymétriques en attribuant des poids spécifiques aux dissimilarités, offrant ainsi une flexibilité et une précision accrues.

Définitions

Pour définir la distance de Gower pour des variables discrètes asymétriques, nous devons d’abord comprendre ce que nous cherchons à mesurer. Imaginons deux objets décrits par une variable discrète asymétrique, comme le niveau de satisfaction (très insatisfait, insatisfait, neutre, satisfait, très satisfait). La dissimilarité entre "très insatisfait" et "très satisfait" n’est pas la même que celle entre "satisfait" et "très satisfait". Nous cherchons donc une mesure qui capture cette asymétrie.

Formellement, soit \(X\) une variable discrète prenant des valeurs dans un ensemble fini \(S = \{s_1, s_2, \ldots, s_k\}\). Nous définissons une matrice de dissimilarité asymétrique \(D\) où chaque entrée \(d_{ij}\) représente la dissimilarité entre les modalités \(s_i\) et \(s_j\). La distance de Gower pour deux objets \(x\) et \(y\) décrits par cette variable est alors donnée par :

\[d_G(x, y) = \frac{1}{k} \sum_{i=1}^{k} d_{x_i y_i}\]

où \(k\) est le nombre de variables, et \(d_{x_i y_i}\) est la dissimilarité entre les modalités \(x_i\) et \(y_i\) de la variable \(i\).

Une autre formulation, plus générale, pour des données mixtes incluant des variables quantitatives et qualitatives, est :

\[d_G(x, y) = \frac{\sum_{i=1}^{k} \delta_i d_{x_i y_i}}{\sum_{i=1}^{k} \delta_i}\]

où \(\delta_i\) est un indicateur égal à 1 si la variable \(i\) est prise en compte, et 0 sinon.

Théorèmes

Pour établir les propriétés de la distance de Gower, nous devons d’abord comprendre ses fondements théoriques. Considérons la matrice de dissimilarité \(D\) comme une mesure de dissimilarité asymétrique. Nous voulons montrer que la distance de Gower ainsi définie satisfait les propriétés métriques fondamentales : non-négativité, identité des indiscernables, symétrie, et inégalité triangulaire.

Soit \(D\) une matrice de dissimilarité asymétrique. La distance de Gower \(d_G\) définie par :

\[d_G(x, y) = \frac{1}{k} \sum_{i=1}^{k} d_{x_i y_i}\]

satisfait les propriétés suivantes :

  1. Non-négativité : \(d_G(x, y) \geq 0\) pour tout \(x, y\).

  2. Identité des indiscernables : \(d_G(x, y) = 0\) si et seulement si \(x = y\).

  3. Symétrie : \(d_G(x, y) = d_G(y, x)\) si et seulement si \(D\) est symétrique.

  4. Inégalité triangulaire : \(d_G(x, z) \leq d_G(x, y) + d_G(y, z)\) pour tout \(x, y, z\).

Preuves

Pour prouver les propriétés métriques de la distance de Gower, nous procédons étape par étape.

Proof.

  1. Non-négativité : Par définition, chaque \(d_{x_i y_i} \geq 0\) car \(D\) est une matrice de dissimilarité. La somme et la division par \(k\) préservent cette propriété. \[d_G(x, y) = \frac{1}{k} \sum_{i=1}^{k} d_{x_i y_i} \geq 0\]

  2. Identité des indiscernables : Si \(x = y\), alors \(d_{x_i y_i} = 0\) pour tout \(i\), donc \(d_G(x, y) = 0\). Réciproquement, si \(d_G(x, y) = 0\), alors chaque \(d_{x_i y_i} = 0\), ce qui implique \(x_i = y_i\) pour tout \(i\), donc \(x = y\).

  3. Symétrie : Si \(D\) est symétrique, alors \(d_{x_i y_i} = d_{y_i x_i}\), donc \(d_G(x, y) = d_G(y, x)\). Réciproquement, si \(d_G(x, y) = d_G(y, x)\), alors \(D\) doit être symétrique.

  4. Inégalité triangulaire : Supposons que \(D\) satisfait l’inégalité triangulaire. Alors, pour tout \(i\), \(d_{x_i z_i} \leq d_{x_i y_i} + d_{y_i z_i}\). En sommant sur \(i\) et en divisant par \(k\), nous obtenons : \[d_G(x, z) \leq d_G(x, y) + d_G(y, z)\]

Propriétés et Corollaires

Nous listons maintenant quelques propriétés importantes de la distance de Gower.

  1. Normalisation : La distance de Gower peut être normalisée pour obtenir une mesure comprise entre 0 et 1. Pour cela, nous divisons chaque \(d_{x_i y_i}\) par la dissimilarité maximale possible pour la variable \(i\).

  2. Poids des variables : Les variables peuvent être pondérées différemment en introduisant des poids \(w_i\) dans la somme. La distance pondérée est alors : \[d_G(x, y) = \frac{\sum_{i=1}^{k} w_i d_{x_i y_i}}{\sum_{i=1}^{k} w_i}\]

  3. Extension aux données manquantes : La distance de Gower peut être étendue pour gérer les données manquantes en excluant simplement les variables avec des valeurs manquantes pour les objets comparés.

Conclusion

La distance de Gower pour variables discrètes asymétriques est un outil puissant et flexible pour l’analyse des données mixtes. Ses propriétés métriques en font une mesure fiable, et son adaptation aux contextes asymétriques permet de capturer la complexité des données réelles. Les extensions possibles, telles que la normalisation et la gestion des données manquantes, en font un choix privilégié pour de nombreuses applications pratiques.
