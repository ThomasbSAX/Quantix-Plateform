Le Test de White : Une Approche Robuste pour l’Hétéroscédasticité

Quantix

Introduction et Motivations

L’analyse des données économiques et financières est souvent confrontée à un défi fondamental : l’hétéroscédasticité. Ce phénomène, où la variance des erreurs n’est pas constante à travers les observations, peut fausser les inférences statistiques et rendre les estimations inefficaces. Le test de White, développé par Halbert White en 1980, offre une solution robuste pour détecter cette hétéroscédasticité.

Le test de White est indispensable dans le cadre des modèles économétriques, où l’hypothèse d’homoscedasticité est souvent trop restrictive. Il permet de valider ou d’invalider cette hypothèse, ce qui est crucial pour choisir les méthodes d’estimation appropriées. Par exemple, en présence d’hétéroscédasticité, les estimateurs par moindres carrés ordinaires ne sont plus efficaces, et des techniques alternatives comme les moindres carrés pondérés ou les méthodes robustes doivent être employées.

Définitions

Pour comprendre le test de White, il est essentiel de définir quelques concepts clés.

Hétéroscédasticité

Supposons que nous avons un modèle linéaire général :

\[y_i = x_i' \beta + u_i\]

où \(y_i\) est la variable dépendante, \(x_i\) est le vecteur des variables explicatives, \(\beta\) est le vecteur des coefficients, et \(u_i\) est le terme d’erreur. L’hétéroscédasticité se produit lorsque la variance de \(u_i\) n’est pas constante pour toutes les observations. Formellement, nous disons qu’il y a hétéroscédasticité si :

\[\text{Var}(u_i | x_i) = \sigma_i^2\]

où \(\sigma_i^2\) dépend de \(x_i\).

Test de White

Le test de White est un test statistique pour détecter l’hétéroscédasticité dans les modèles de régression. Il repose sur une régression auxiliaire des carrés des résidus au carré sur les variables explicatives et leurs carrés. Formellement, le test de White peut être décrit comme suit :

Soit \(\hat{u}_i\) les résidus obtenus à partir de l’estimation du modèle initial. Le test de White consiste à estimer le modèle auxiliaire :

\[\hat{u}_i^2 = x_i' \gamma + z_i' \delta + v_i\]

où \(z_i\) est un vecteur contenant les carrés et les produits croisés des variables explicatives. Le test statistique est basé sur le test de l’hypothèse nulle que tous les coefficients \(\gamma\) et \(\delta\) sont nuls.

Théorèmes

Théorème de White

Le théorème central du test de White peut être énoncé comme suit :

Théorème 1 (White). Soit \(y_i = x_i' \beta + u_i\) un modèle linéaire général, où \(u_i\) est un terme d’erreur. Supposons que les conditions habituelles de régression linéaire sont satisfaites, sauf l’hypothèse d’homoscedasticité. Alors, le test de White pour l’hétéroscédasticité est asymptotiquement distribué selon une loi du chi-deux avec \(k\) degrés de liberté, où \(k\) est le nombre de restrictions testées.

Démonstration. La démonstration repose sur les propriétés asymptotiques des tests de régression. Sous l’hypothèse nulle d’homoscedasticité, les résidus \(\hat{u}_i\) sont indépendants et identiquement distribués avec une variance constante. Le test de White consiste à régresser les résidus au carré sur les variables explicatives et leurs carrés.

En utilisant le théorème de la limite centrale, nous pouvons montrer que le test statistique suit asymptotiquement une loi du chi-deux. Plus précisément, sous l’hypothèse nulle, le test statistique \(T\) est donné par :

\[T = n R^2\]

où \(n\) est le nombre d’observations et \(R^2\) est le coefficient de détermination du modèle auxiliaire. Sous l’hypothèse nulle, \(T\) suit une loi du chi-deux avec \(k\) degrés de liberté.

Preuves

Justification du Test de White

Pour justifier le test de White, nous devons montrer que sous l’hypothèse nulle d’homoscedasticité, le test statistique suit une loi du chi-deux. Considérons le modèle auxiliaire :

\[\hat{u}_i^2 = x_i' \gamma + z_i' \delta + v_i\]

Sous l’hypothèse nulle, \(\gamma = 0\) et \(\delta = 0\). Les résidus \(v_i\) sont alors indépendants et identiquement distribués avec une variance constante. En utilisant le théorème de la limite centrale, nous pouvons montrer que le test statistique \(T\) est asymptotiquement distribué selon une loi du chi-deux.

Plus précisément, nous avons :

\[T = n R^2\]

où \(R^2\) est le coefficient de détermination du modèle auxiliaire. Sous l’hypothèse nulle, \(R^2\) est asymptotiquement distribué selon une loi du chi-deux avec \(k\) degrés de liberté.

Propriétés et Corollaires

Propriétés du Test de White

Le test de White possède plusieurs propriétés importantes :

  1. Consistance. Le test de White est consistant pour détecter l’hétéroscédasticité. Cela signifie que sous l’hypothèse alternative, la probabilité de rejeter l’hypothèse nulle tend vers 1 lorsque le nombre d’observations tend vers l’infini.

  2. Robustesse. Le test de White est robuste à la présence de multicolinéarité dans les variables explicatives. Cela signifie que même si les variables explicatives sont fortement corrélées, le test reste valide.

  3. Asymptoticité. Le test de White est asymptotiquement distribué selon une loi du chi-deux. Cela signifie que pour un grand nombre d’observations, la distribution du test statistique est bien approchée par une loi du chi-deux.

Corollaires

Corollaire 1. Si le test de White rejette l’hypothèse nulle d’homoscedasticité, alors les estimateurs par moindres carrés ordinaires ne sont plus efficaces. Des méthodes alternatives comme les moindres carrés pondérés ou les méthodes robustes doivent être employées.

Corollaire 2. Si le test de White ne rejette pas l’hypothèse nulle d’homoscedasticité, alors les estimateurs par moindres carrés ordinaires sont efficaces et peuvent être utilisés pour l’estimation du modèle.

Conclusion

Le test de White est un outil puissant pour détecter l’hétéroscédasticité dans les modèles de régression. Il offre une solution robuste et asymptotiquement valide pour tester l’hypothèse d’homoscedasticité. En comprenant les définitions, les théorèmes et les propriétés du test de White, les chercheurs peuvent mieux analyser leurs données et choisir les méthodes d’estimation appropriées.
