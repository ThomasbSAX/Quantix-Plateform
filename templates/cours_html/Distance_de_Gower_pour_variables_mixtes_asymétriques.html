La distance de Gower pour variables mixtes asymétriques

Quantix

2026-01-11

Introduction et Motivations

La distance de Gower émerge dans un contexte où les données présentent des caractéristiques mixtes et asymétriques. Historiquement, cette notion a été introduite pour répondre à des besoins spécifiques en analyse de données multivariées, où les variables ne sont pas toutes de même nature. En effet, dans de nombreux domaines tels que la biologie, l’économie ou les sciences sociales, il est courant de se retrouver confronté à des jeux de données contenant à la fois des variables quantitatives et qualitatives, voire ordinales.

L’asymétrie des données ajoute une couche de complexité supplémentaire. Les méthodes classiques de distance, comme la distance euclidienne ou la distance de Manhattan, ne sont pas adaptées pour traiter ces types de données. La distance de Gower propose une solution élégante en normalisant les contributions des différentes variables, qu’elles soient quantitatives ou qualitatives.

Cette distance est indispensable dans le cadre de l’analyse des données multivariées, où la compréhension des relations entre les individus et les variables est cruciale. Elle permet de comparer des objets décrits par des variables de nature différente, tout en tenant compte de l’asymétrie potentielle des données.

Définitions

Pour définir la distance de Gower, commençons par comprendre ce que nous cherchons à mesurer. Nous voulons une distance qui soit capable de comparer des individus décrits par des variables de nature différente, tout en tenant compte de l’échelle et de la nature de chaque variable. La distance doit également être insensible aux asymétries dans les données.

Formellement, la distance de Gower entre deux individus \(i\) et \(j\) est définie comme suit :

Soit \(X\) une matrice de données de taille \(n \times p\), où \(n\) est le nombre d’individus et \(p\) est le nombre de variables. Pour chaque variable \(k\), définissons une fonction de distance \(d_k\) adaptée à sa nature.

Pour les variables quantitatives, la distance est normalisée par l’écart interquartile : \[d_k(x_{ik}, x_{jk}) = \frac{|x_{ik} - x_{jk}|}{Q_k^3 - Q_k^1}\] où \(Q_k^1\) et \(Q_k^3\) sont les premier et troisième quartiles de la variable \(k\).

Pour les variables qualitatives, la distance est binaire : \[d_k(x_{ik}, x_{jk}) = \begin{cases} 0 & \text{si } x_{ik} = x_{jk} \\ 1 & \text{sinon} \end{cases}\]

Pour les variables ordinales, la distance est normalisée par le nombre de catégories : \[d_k(x_{ik}, x_{jk}) = \frac{|x_{ik} - x_{jk}|}{\text{nombre de catégories}}\]

La distance de Gower globale est alors définie comme la moyenne des distances normalisées sur toutes les variables : \[d_{Gower}(i, j) = \frac{\sum_{k=1}^p w_k d_k(x_{ik}, x_{jk})}{\sum_{k=1}^p w_k}\] où \(w_k\) est le poids de la variable \(k\), souvent pris égal à 1 si toutes les variables sont considérées comme égales.

Théorèmes

Pour comprendre les propriétés de la distance de Gower, examinons quelques théorèmes importants.

La distance de Gower satisfait les propriétés suivantes :

  1. Identité des indifférents : \(d_{Gower}(i, j) = 0\) si et seulement si \(i = j\).

  2. Symétrie : \(d_{Gower}(i, j) = d_{Gower}(j, i)\).

  3. Inégalité triangulaire : \(d_{Gower}(i, j) + d_{Gower}(j, k) \geq d_{Gower}(i, k)\).

Proof. Pour prouver ces propriétés, nous procédons comme suit :

1. Identité des indifférents : Par définition, si \(i = j\), alors \(d_k(x_{ik}, x_{jk}) = 0\) pour toutes les variables \(k\). Ainsi, \(d_{Gower}(i, j) = 0\).

2. Symétrie : La symétrie découle directement de la définition des distances \(d_k\), qui sont toutes symétriques.

3. Inégalité triangulaire : Pour chaque variable \(k\), l’inégalité triangulaire est satisfaite pour la distance \(d_k\). En prenant la moyenne pondérée, l’inégalité triangulaire est préservée. ◻

Preuves

Pour illustrer les preuves, prenons l’exemple de la propriété d’identité des indifférents.

Proof. Soit \(i = j\). Alors, pour chaque variable \(k\), nous avons \(x_{ik} = x_{jk}\). Par définition des distances \(d_k\), nous avons : \[d_k(x_{ik}, x_{jk}) = 0\]

En prenant la moyenne pondérée sur toutes les variables, nous obtenons : \[d_{Gower}(i, j) = \frac{\sum_{k=1}^p w_k \cdot 0}{\sum_{k=1}^p w_k} = 0\]

Réciproquement, si \(d_{Gower}(i, j) = 0\), alors pour chaque variable \(k\), nous devons avoir \(d_k(x_{ik}, x_{jk}) = 0\). Cela implique que \(x_{ik} = x_{jk}\) pour toutes les variables \(k\), donc \(i = j\). ◻

Propriétés et Corollaires

La distance de Gower possède plusieurs propriétés intéressantes, que nous listons ci-dessous :

  1. Normalisation : La distance de Gower est normalisée par les caractéristiques des variables, ce qui permet de comparer des individus décrits par des variables de nature différente.

  2. Robustesse aux asymétries : La distance de Gower est robuste aux asymétries dans les données, grâce à l’utilisation de mesures de dispersion adaptées.

  3. Flexibilité : La distance de Gower peut être adaptée en ajustant les poids \(w_k\) des variables, permettant ainsi de donner plus ou moins d’importance à certaines variables.

Proof. Pour prouver ces propriétés, nous procédons comme suit :

1. Normalisation : La normalisation est assurée par les fonctions de distance \(d_k\), qui sont définies en fonction des caractéristiques spécifiques de chaque variable.

2. Robustesse aux asymétries : L’utilisation des quartiles pour les variables quantitatives et la normalisation par le nombre de catégories pour les variables ordinales rend la distance insensible aux asymétries.

3. Flexibilité : La flexibilité est assurée par les poids \(w_k\), qui peuvent être ajustés en fonction des besoins de l’analyse. ◻

Conclusion

La distance de Gower est un outil puissant pour l’analyse des données multivariées, permettant de comparer des individus décrits par des variables de nature différente. Ses propriétés métriques et sa robustesse aux asymétries en font un choix privilégié dans de nombreux domaines d’application.
