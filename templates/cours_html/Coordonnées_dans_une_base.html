Coordonnées dans une base

Quantix

2026-01-11

Introduction et Motivations

L’étude des coordonnées dans une base est un pilier fondamental de l’algèbre linéaire. Cette notion permet de représenter les vecteurs d’un espace vectoriel de manière concrète et manipulable. Historiquement, l’introduction des coordonnées cartésiennes par René Descartes au XVIIème siècle a marqué un tournant dans la géométrie analytique. Aujourd’hui, cette idée s’étend à des espaces vectoriels abstraits de dimension quelconque.

Les coordonnées dans une base permettent de résoudre des problèmes variés : résolution d’équations linéaires, optimisation, transformation géométrique, et bien plus encore. Elles sont indispensables dans de nombreux domaines scientifiques, comme la physique, l’ingénierie, et l’informatique.

Définitions

Pour introduire les coordonnées dans une base, commençons par comprendre ce que nous cherchons à obtenir. Considérons un espace vectoriel \(E\) de dimension \(n\). Nous voulons représenter les vecteurs de cet espace par des combinaisons linéaires d’un ensemble fixe de \(n\) vecteurs, appelés base. Les coordonnées d’un vecteur par rapport à cette base sont les coefficients de cette combinaison.

Soit \(E\) un espace vectoriel de dimension \(n\). Une base de \(E\) est un ensemble \(\{e_1, e_2, \ldots, e_n\}\) de vecteurs de \(E\) tels que :

  1. Les vecteurs \(e_1, e_2, \ldots, e_n\) sont linéairement indépendants.

  2. Tout vecteur \(v \in E\) peut s’écrire comme une combinaison linéaire des \(e_i\), c’est-à-dire qu’il existe des scalaires \(\lambda_1, \lambda_2, \ldots, \lambda_n\) tels que : \[v = \sum_{i=1}^n \lambda_i e_i.\]

Soit \(\{e_1, e_2, \ldots, e_n\}\) une base de l’espace vectoriel \(E\). Les coordonnées du vecteur \(v \in E\) dans cette base sont les scalaires \(\lambda_1, \lambda_2, \ldots, \lambda_n\) tels que : \[v = \sum_{i=1}^n \lambda_i e_i.\] On note ces coordonnées par un vecteur colonne : \[[v]_{\mathcal{B}} = \begin{pmatrix} \lambda_1 \\ \lambda_2 \\ \vdots \\ \lambda_n \end{pmatrix},\] où \(\mathcal{B} = \{e_1, e_2, \ldots, e_n\}\).

Théorèmes

Pour mieux comprendre les coordonnées dans une base, examinons quelques théorèmes clés. Commençons par le théorème qui garantit l’existence et l’unicité des coordonnées d’un vecteur dans une base.

Soit \(E\) un espace vectoriel de dimension \(n\), et soit \(\{e_1, e_2, \ldots, e_n\}\) une base de \(E\). Pour tout vecteur \(v \in E\), il existe un unique ensemble de scalaires \(\lambda_1, \lambda_2, \ldots, \lambda_n\) tels que : \[v = \sum_{i=1}^n \lambda_i e_i.\]

Preuves

Pour prouver le théorème d’existence et d’unicité des coordonnées, nous allons utiliser les propriétés des bases et des combinaisons linéaires.

Proof. Soit \(E\) un espace vectoriel de dimension \(n\), et soit \(\{e_1, e_2, \ldots, e_n\}\) une base de \(E\). Soit \(v \in E\).

Existence : Puisque \(\{e_1, e_2, \ldots, e_n\}\) est une base de \(E\), par définition, tout vecteur \(v \in E\) peut s’écrire comme une combinaison linéaire des \(e_i\). Il existe donc des scalaires \(\lambda_1, \lambda_2, \ldots, \lambda_n\) tels que : \[v = \sum_{i=1}^n \lambda_i e_i.\]

Unicité : Supposons qu’il existe deux ensembles de scalaires \(\lambda_1, \lambda_2, \ldots, \lambda_n\) et \(\mu_1, \mu_2, \ldots, \mu_n\) tels que : \[v = \sum_{i=1}^n \lambda_i e_i = \sum_{i=1}^n \mu_i e_i.\] En soustrayant ces deux égalités, nous obtenons : \[\sum_{i=1}^n (\lambda_i - \mu_i) e_i = 0.\] Puisque les vecteurs \(e_1, e_2, \ldots, e_n\) sont linéairement indépendants, il s’ensuit que : \[\lambda_i - \mu_i = 0 \quad \text{pour tout } i = 1, 2, \ldots, n.\] Ainsi, \(\lambda_i = \mu_i\) pour tout \(i\), ce qui prouve l’unicité des coordonnées. ◻

Propriétés et Corollaires

Examinons quelques propriétés importantes des coordonnées dans une base.

  1. Changement de base : Soit \(E\) un espace vectoriel, et soit \(\mathcal{B} = \{e_1, e_2, \ldots, e_n\}\) et \(\mathcal{B}' = \{f_1, f_2, \ldots, f_n\}\) deux bases de \(E\). Pour tout vecteur \(v \in E\), les coordonnées de \(v\) dans la base \(\mathcal{B}'\) peuvent être exprimées en fonction de ses coordonnées dans la base \(\mathcal{B}\) par une matrice de passage.

  2. Opérations linéaires : Les opérations linéaires (addition, multiplication par un scalaire) sur les vecteurs peuvent être exprimées directement en termes de leurs coordonnées dans une base. Par exemple, si \(v\) et \(w\) sont deux vecteurs de \(E\), alors : \[[v + w]_{\mathcal{B}} = [v]_{\mathcal{B}} + [w]_{\mathcal{B}},\] \[[\alpha v]_{\mathcal{B}} = \alpha [v]_{\mathcal{B}},\] où \(\alpha\) est un scalaire.

  3. Produit matriciel : Si \(E\) et \(F\) sont deux espaces vectoriels de dimensions respectives \(n\) et \(m\), et si \(\mathcal{B} = \{e_1, e_2, \ldots, e_n\}\) et \(\mathcal{C} = \{f_1, f_2, \ldots, f_m\}\) sont des bases de \(E\) et \(F\), alors toute application linéaire \(T : E \to F\) peut être représentée par une matrice \(A\) telle que : \[[T(v)]_{\mathcal{C}} = A [v]_{\mathcal{B}},\] pour tout \(v \in E\).

Conclusion

Les coordonnées dans une base sont un outil puissant et fondamental en algèbre linéaire. Elles permettent de représenter les vecteurs de manière concrète et manipulable, facilitant ainsi la résolution de nombreux problèmes. Les théorèmes d’existence et d’unicité des coordonnées, ainsi que les propriétés des changements de base et des opérations linéaires, sont essentiels pour comprendre et utiliser efficacement cette notion.
