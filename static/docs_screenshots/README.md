# Captures d’écran pour la documentation

Placez ici des captures d’écran (PNG/JPG) utilisées par `templates/docs.html`.

Objectif : illustrer les boutons et étapes clés (import, nettoyage, plots, graph studio, labelling).

## Noms attendus (proposés)
- `dashboard_import.png`
- `dashboard_clean.png`
- `lab_plotstudio.png`
- `graph_studio.png`
- `labelling_markers.png`

Vous pouvez utiliser d’autres noms : mettez ensuite à jour les `src="/static/docs_screenshots/..."` dans `templates/docs.html`.

## Conseils
- Largeur recommandée : 1400–1800 px
- Format : PNG
- Floutez/masquez toute donnée sensible
